Macintosh VPN

Virtual Private Networking (VPN) software helps secure your computer when connecting remotely to a network. Traffic to and from your computer is encrypted.

To connect the the CWRU network with VPN, download Cisco AnyConnect from the Software Center.

Install the AnyConnect Client

  1. Download Cisco AnyConnect VPN Client installation file from the Software Center.
  2. Launch the installation program if it does not begin automatically.
  3. Click Connect to begin.
  4. Click Continue to display the Software License Agreement.
  5. Review the Software License Agreement and click Continue.
  6. Choose Agree to continue or Disagree to cancel the installation.
  7. UNCHECK the Web Security, Diagnostics and Reporting Tool and Posture options.
    NOTE: These options are not supported on the CWRU network.
  8. Click Continue and then Install. Do not change the default Install Location.
  9. Enter computer administrator name and password into Name and Password fields.
  10. Click OK and then Close after the confirmation screen appears.

Configure the VPN connection and Connect

  1. Press ⌘ + spacebar to launch Spotlight Search.

  2. Enter “vpn” in the search box.

  3. Choose the Cisco AnyConnect VPN Client and click to launch it.

  4. Enter “vpn.case.edu” in connect to field.
  5. Click Select.
  6. Enter CWRU Network ID and password when prompted.
  7. If using two-factor authentication provided by CaseDuo, enter Duo passcode in the “Second Password” field.
  8. Otherwise, enter any text in the “Second Password” field.
  9. Click Connect and then click Accept to agree to the Case Acceptable Use Policy (AUP).
  10. The connection will minimize to the menu bar.

  11. Right-click the AnyConnect icon. Select Options > Keep in Dock.

    NOTE: Your “group” name may vary. Most users will see the group name “Case”, which is the default VPN profile.

Disconnecting and reconnecting

  1. Click the AnyConnect icon in the Dock to open the application.

  2. Select vpn.case.edu from the dropdown box, and click Connect to reestablish VPN connection.

    NOTE: An orange lock symbol overlays the AnyConnect icon when connection is confirmed.

  3. To disconnect VPN, right-click the AnyConnect icon and select Disconnect.
