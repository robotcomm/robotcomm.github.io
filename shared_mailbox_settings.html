Shared mailboxes

Access your shared mailbox on or off-campus from any telephone.

Settings

  1. Dial 368-8744. NOTE: Before 01/07/2015, dial 9, then +1 (216) 368-8744
  2. Enter your ID and # (personal mailbox number is provided by ITS)

  3. Enter your PIN (default, 10900)

    First time users

  4. Follow the prompts to record name, greeting and change PIN

    Returning user

  5. Enter 11-digit shared mailbox number (“extension of the call handler”) and #
  6. Select option to set a new greeting change current set greeting

Messages (any phone)

  1. Dial +1 (216) 754-3800
  2. Enter your voicemail box number
  3. Enter your PIN (default, 10900)

Messages (desk phone)

  1. Press the Messages key
  2. Press *
  3. Enter your voicemail box number
  4. Enter your PIN (default, 10900)


Please note: If you are accessing shared mailbox settings for the first time after 01/07/2015, use the default PIN, 10900. To protect your privacy, use the instructions at case.edu/its/uc to set a new PIN immediately.
