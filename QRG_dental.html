Voicemail Quick Start Guide

How do I set or change my voicemail PIN?

  1. Navigate to unity-vm1.case.edu/inbox/.
  2. Enter CWRU network ID and password when prompted.
  3. Click Settings to open the “Messaging Assistant”.
  4. Select Passwords menu and then Change PIN.
  5. Type a new PIN in the box labelled New PIN.
  6. Type new PIN again in the next box to confirm.
  7. Click the Save button at the BOTTOM of the page.
  8. When confirmation message appears, “Your changes were successfully saved”, new PIN has been applied.

NOTE: default PIN is 10900

How do I use a computer to set Greetings?

Available greeting options

The following greeting types are available.

  • Alternate (disabled by default - when enabled, overrides all other greetings)
  • Busy (disabled by default)
  • Internal (disabled by default)
  • Closed (disabled by default)
  • Standard (enabled default)
  • Holiday (disabled by default)

NOTE:
When enabled, each greeting type may use a system default greeting, personal recording or nothing at all. However, only one greeting type may be enabled at a time for a selected telephone number.

Enable a greeting

  1. Navigate to unity-vm1.case.edu/inbox/.
  2. Enter CWRU network ID and password when prompted.
  3. Click Settings to open the “Messaging Assistant”.
  4. Select Greetings menu and then View Greetings.
    1 Click the greeting name (look in the column marked “Greeting”).
  5. Choose one status option,
    • Disabled
    • Enabled with no end date and time
    • Enabled until …
      1. Choose end date (Month, Day, Year)
      2. Choose end time (HH:MM AM/PM)
  6. Choose one Greeting Audio option,
    • My Personal Recording (record, re-record or review recorded greeting, if needed)
    • System default greeting
    • Nothing
  7. If using a personal recording, also select Callers See, “My Personal Recording”.
  8. Click the Save button at the BOTTOM of the page.
  9. When confirmation message appears, “Your changes were successfully saved”, greeting changes have been applied.

Record a greeting

  1. Navigate to unity-vm1.case.edu/inbox/.
  2. Enter CWRU network ID and password when prompted.
  3. Click Settings to open the “Messaging Assistant”.
  4. Select Greetings menu and then View Greetings.
    1 Click the greeting name (look in the column marked “Greeting”).

  5. In the menu section marked “Callers Hear” choose the the option My Personal Recording. NOTE: A greeting need not be enabled for a user to record a message for it.

    Set recording options

    1. Click “Play/Record” NOTE: this option requires Java browser plugins. The current version (v8 update 25) is available from Oracle.
    2. Leave the default “Playback” option (Use Computer).
    3. Select record option Use Computer from the dropdown menu.

    Record or re-record greeting

    1. Click the record button (represented by a red circle).
    2. System will call your default telephone number and prompt recording.
    3. Press pause button (represented by two vertical bars) to pause (and resume) recording.
    4. Press the stop button (represented by a black square) to stop recording.

    Review recorded greeting

    1. Click the play button (represented by a triangle pointing right) to hear recorded name.
  6. Select Callers See, “My Personal Recording”.
  7. Click the Save button at the BOTTOM of the page.
  8. Confirmation message appears, “Your changes were successfully saved”.
  9. When confirmation message appears, “Your changes were successfully saved”, greeting changes have been applied.
