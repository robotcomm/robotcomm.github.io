Identity Finder works like an anti-virus product. Client software installs locally, inspects storage media for data matching certain pre-defined criteria, and logs the activity with a management console operated by the Information Security Office. When pattern matches are found (e.g., potential social security numbers, credit card numbers) Identity Finder presents several options to remove the data:

  • Destroy the file (Shred)
  • Redact information from the file (Scrub)
  • Prepare the file for move to a secure location (Quarantine)
  • Safeguard the file using proprietary encryption (Secure)

In the case of a 'false positive' match users may also instruct Identity Finder to whitelist a file or location (Ignore).

Install Identity Finder

Use these directions to download and install Identity Finder.

Windows

  1. Sign in to the Software Center and agree to the End User License Agreement.
  2. Locate Identity Finder.
  3. Click the Windows Download button.
  4. A new page opens. Scroll to the bottom of the page and click Download Identity Finder.
  5. Choose a memorable 'save-to' location (e.g., your desktop) and click Save.
  6. Open the downloaded item and click Run when prompted.
  7. Follow the prompts to install Identity Finder on your computer.
  8. Open the Identity Finder application.
  9. Choose your school or department from the list that appears and click Select.
  10. Refer to the linked Identity Finder manual for next steps.

Mac OS

  1. Sign in to the Software Center and agree to the End User License Agreement.
  2. Locate Identity Finder.
  3. Click the Macintosh Download button.
  4. A new page opens. Scroll to the bottom of the page and click Download Identity Finder.
  5. Choose a memorable 'save-to' location (e.g., your desktop) and click Save.
  6. Open the downloaded item and double-click the package installer to begin.
  7. Follow the prompts to install Identity Finder on your computer.
  8. Launch the Identity Finder application.
  9. Choose your school or department from the list that appears and click Select.
  10. Refer to the linked Identity Finder manual for next steps.

Linux

Although Identity Finder is not available for Linux users through the Software Center, the vendor does offer an endpoint compatible with various Linux distributions. Please contact your management center for licensing help.

Perform a basic scan

Use these instructions to scan your computer for sensitive information. Before you begin, backup your data. 'Shredded' files are completely destroyed, and cannot be recovered.

Perform a basic scan in weekly or monthly intervals.

  1. After installation, Identity Finder will open and present the Profile Sign In screen.
  2. Choose an Identity Finder Profile password and enter it in the Enter Password field. Enter it again in the Confirm Password field.



  3. The Identity Finder application window appears.
    NOTE: Do not create or change any settings in the application. Appropriate settings are pre-defined by ITS.
  4. Click Start.
  5. While Identity Finder scans your machine you may continue with other tasks.
  6. When the scan is complete, the Search Results Summary will display search locations any matches found within them. You are required to review any matches found during the scan.
  7. Click Save As to save your scan results and review them at your convenience.
    NOTE: The scan results file is secured by your Identity Finder Profile Password.

Review scan results

  1. Open scan results file, or continue from Search Results Summary.
  2. Click Advanced.
  3. Multiple matches within a single file are displayed in a drill-down list.
  4. Click the first row or list entry in the results screen.
  5. A preview of the file containing restricted data displays file contents with restricted data highlighted.



  6. Review each individual result and make an appropriate handling decision (i.e., Shred, Scrub, Secure, Quarantine, Recycle or Ignore).
  7. When multiple rows can be addressed with the same action,
    1. Select each row by clicking in the checkbox to its left
    2. Choose and execute the appropriate action
    3. All selected rows will be handled with that action

  8. To scrub (redact sensitive information) a single file with be multiple matches,

    1. Select the row that contains the drill-down list
    2. Click Scrub button




    3. All restricted data in the single file will be removed from the document and replaced with X's.

  9. If a file contains sensitive (restricted) data that must be retained, contact Information Security to discuss available secure storage options. This data may be subject to regulatory controls.
    NOTE: Per university policy, most faculty and staff are not authorized to maintain Restricted data unless protected by Tier III Controls.
  10. The Secure, Quarantine and Recycle options should only be used when specifically directed to do so by Information Security or its designated proxy.
  11. Click Save in the upper left corner of the application window mark your progress and continue at another time.
  12. When all restricted data found by the scan has been addressed, the message 'There are no items to show' will appear in the search results pane.
  13. Run a scan once more to report any additional results. If nothing is found, the immediate task is complete.

Sort and filter scan results

Sorting and filtering can be used by end-users to better identify and eliminate or secure the data that is most vulnerable on a machine.

  1. Perform a basic scan. Close the Status Window when complete.
  2. Click Filter Results.
  3. Search filters appear above the scan results.
  4. Click Search For: All Identity Types.
  5. Choose an identity to see only the matching scan results.
  6. Click the Search For: All Location Types.
  7. Choose a location to see scan results that are only in that location.
  8. To remove filter settings, click Filter Results.
  9. Sorted by File Type, Location, Date Modified, or any other column shown in the Results Columns menu.
  10. Click column header to sort by desired criteria.
  11. Any multiple match row can be expanded/collapsed using the "plus" or "minus" sign to its left.
  12. To collapse all drill down menus click Collapse All Rows.



  13. To expand all multiple match rows, click Expand All Rows.

Modify results view

The Configuration tab contains settings for how the Results View and Preview Pane are displayed. To change the layout of the results screen, click the Configuration tab.

Element Action
Results View Dropdown menu contains a list of the columns that can appear in the scan results. Columns may be added or removed.

  1. Select a column without a checkmark to add it to search results
  2. Unselect a column with a checkmark to remove it from search results
Preview Pane
  1. To move the preview pane; click panel header and drag panel into desired location
  2. Close Properties Pane completely by clicking Properties Pane button
Default View To restore the default view click the Reset View button.
