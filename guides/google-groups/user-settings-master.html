While each group is created with a specific set of attributes, many of the default settings can be modified by the group’s manager. These customizable attributes are accessed from the menu appearing on the left side of the group management screen, and are divided into four categories:

  • Settings - controls group appearance and behavior with regard to members
  • Permissions - determines the degree of control a user or role has with regard to group settings
  • Roles - controls settings and permissions for a named class of users
  • Information - controls how the group interacts with mail servers and appears on the web

Wildcard Search Values

Standard wildcard characters may be used in searches. Searches that return a large number records (e.g., Network ID equals *) will take a long time to complete and may cause the browser to time out.

Wildcard searches are most useful when you do not know if a name has an unusual spelling, pronunciation, or is long and may have been shortened in some records (for example Shannon Huntington Smythe-Whitely => Shannon Huntington). A search like Full name equals Shannon *huntington* would reduce the selection set considerably but still produce useful results.
