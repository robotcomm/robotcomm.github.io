Technology Enhanced Classroom (TEC) Classifications

For more detail, visit http://www.case.edu/its/tec/.

Level 1

  • ceiling mounted video projection system
  • projection screen
  • interface plate for network, computer, video and audio sources
  • remote control panel to turn the system on and off, select input sources, adjust audio volume
  • wall-mounted program speakers

Level 1 HD

  • includes components and features of Level 1
  • native 720p (HD) projector, 16x9 format

Level 1.5

  • includes components and features of Level 1
  • equipment rack doubles as teach station that features:
    • built-in PC
    • retractable keyboard and mouse tray
    • LCD monitor
  • document camera
  • rack mounted VCR/DVD player

Level 2

  • includes components and features of Level 1
  • equipment rack doubles as teach station that features:
    • height-adustable surface
    • built-in PC
    • retractable keyboard and mouse tray
    • built-in LCD monitor
  • document camera
  • rack mounted VCR/DVD player
  • remote control panel controls basic media player functions (start/pause/stop/eject)
  • browser-based controls for advanced media player functions (chapter, menu search)
  • remote room technical support
  • dedicated telephone for room support

Level 2 HD

  • includes components and features of Level 2
  • native 720p (HD) projector, 16x9 format
  • 15" LCD touch panel (confidence monitor and presentation control center)
  • Blu-Ray player

Level 3

  • includes components and features of Level 2 HD
  • equipped for videoconference
