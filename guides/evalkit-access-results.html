Using EvalKIT Survey Results

You may report results for each course individually or generate a report containing results for all sections. Reports are available for 7 days after being created (view, print or download).

Access results

  1. Type “case.evaluationkit.com” and sign in using your CWRU network ID and password

  2. Click Results > Project Results

  3. Click the document icon in “View” column

Report results for an individual course

  1. Click View in the “Report” column

  2. Choose one of three report types:


    PDF report
    descriptive statistics, does not include student comments

    PDF + Text report
    descriptive statistics, does include student comments

    Raw Data
    Excel raw data file

  3. NOTE: Results are not generated for courses with fewer than 4 student respondents

Report results for multiple course sections

  1. Select the check box next to “course code” to select all courses. Or, select multiple sections individually.

    Note: Adjust number of results per page (“# records”) to verify that you have selected all courses.

  2. Click Generate Batch Report

  3. Click GO to produce a report for selected sections.

  4. Results are not generated for courses with fewer than 4 student respondents

  5. You will receive an email when requested report is ready. The report is available through the email alert for 7 days. After the report expires, you must generate a new report.

Please contact the Office of the Provost with policy questions. Or if you experience difficulty accessing evaluation results, call (216) 368-HELP.
