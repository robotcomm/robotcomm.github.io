Box Secure Storage

Box.com secure storage is the most appropriate choice for handling sensitive data online at Case Western Reserve (CWRU). Data stored using Box must be classified as Internal Use Only or Restricted.

Find out more about features that distinguish Box from other cloud storage services.

Additional considerations

Case Western Reserve (CWRU) has contractual agreements with Box and Google Apps for Education protecting the confidentiality and integrity of information stored using these products.

Box.com is intended for hosting information with high confidentiality requirements (Restricted information). Google Drive (a Google Apps for Education product) is available to most CWRU users and is approved for use with Internal Use Only and Public information.

CWRU will not implement security or privacy measures that control personal Dropbox accounts or the data stored using Dropbox services. CWRU strongly discourages the use of Dropbox to manage or host university data.

Participation in the Box Pilot program is by invitation only. More information concerning the program is available from your Chief Technology Officer or Information Technology Services.
