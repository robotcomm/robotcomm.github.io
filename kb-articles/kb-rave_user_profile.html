Rave Alert User Profile Setup

These instructions explain how to set up your Rave user profile to receive text, voice and email alerts.

Procedure

To begin, access your Rave profile at www.getrave.com

Step

Action

1.

From the Single Sign-On page, enter your CWRU Network ID and password and then click the Login button.

 

 

Step

Action

2.

If the Rave Mobile Safety log in page appears, type in "case" to bring up CWRU on the drop down list.

3.

Click the Case Western Reserve University menu entry to select it.

4.

Click the Continue button.

 

Step

Action

5.

The Terms of Use page appears. You can use the scroll bar to the right of the text to view the service agreement information.

6.

Click the I have read and agree to the Rave Terms of Use check box to indicate that you agree to the Rave terms of service.

 

7.

Click the Submit button.

 

Step

Action

8.

Your RaveAlert profile page will appear. You can edit profile details from the My Account tab.

9.

Click the Edit button next to your name to begin.

 

Step

Action

10.

From the User Profile page you can edit the First/Last Name fields and select either English or Spanish as your preferred alert language from the drop down list.

11.

When finished, click the Submit button.

 

Step

Action

12.

You will receive a banner message stating: Success: User Profile successfully updated.

 

Step

Action

13.

The Rave system will accept up to 3 mobile phone numbers to which Short Message Service (SMS) text alerts can be sent.

If you have not previously registered a mobile phone, you will receive the following banner message: To take full advantage of this service, you must add at least one Mobile Contact.

14.

Click the Register Phone button that appears on the right of the banner message.

 

Step

Action

15.

Enter your 10-Digit Mobile Number in the field provided.

 

16.

The Enable Voice Message Delivery option is selected by default and will allow you receive voice message alerts in addition to text messages.

If you wish to receive text messages only, then click the check box to deselect this option.

17.

Click the Continue button.

 

Step

Action

18.

Select your cell provider from the Confirm your carrier drop down list.

 

Step

Action

19.

Click the Continue button when finished.

 

Step

Action

20.

Check your mobile phone for a text message from 226787 or 67283:

cwru: Your confirmation code is 8717. Reply w/ Help for Help, Stop to Cancel. Msg & data rates may apply. Info: getrave.com.

Enter your Confirmation code into the field provided.

Note: Failure to receive a confirmation code during mobile registration may be due to a premium messaging block placed by your carrier. If you do not receive the code, please contact your carrier and ask to have messages from short codes 67283 and 226787 delivered to your mobile device.

 

Step

Action

21.

You can use the Resend Confirmation Code button to try again (if needed).

 

Step

Action

22.

When your mobile number has been succesfully added to the contacts list, click the Done button.

 

 

Step

Action

23.

Your mobile phone number will now show as Confirmed on the My Account page.

 

24.

Use the +Add button to add more up to 2 more numbers.

 

Step

Action

25.

Use the Test button to generate a test text message from the Rave system.

26.

Use the Edit button to edit existing numbers.

27.

Use the Delete button to remove a number.

 

Step

Action

28.

Note for Students: If you do not have a cellular contact number, you can enter 999-999-9999.

You must use the Cancel button from the confirmation screen to skip the confirmation process since there is no number to verify.

Even though the 999-999-9999 number is not confirmed, it will still clear the pop-up message you receive when logging in to SIS.

 

Step

Action

29.

You can use the Voice Only Line Contact function to add a contact number to receive Voice Messages Only.

Click the +Add button.

 

Step

Action

30.

Enter the Phone Number in field provided.

31.

Click the Save button.

 

Note: There is no confirmation process for Voice Only Line Contact numbers.

 

Step

Action

32.

Rave will accept up to 3 email addresses. We recommend that you add at least one address that is not affiliated with CWRU.

Note: Please do not use CWRU alias email addresses for this purpose.

33.

The Registration email field displays your CWRU email address.

34.

Click the +Add button.

 

Step

Action

35.

Enter your off-campus email address into the Preferred Email contact field.

36.

Click the Save button.

 

Step

Action

37.

You can use the Test button to send a test message to your Registration or Preferred email accounts.

38.

Click the Edit button to edit an existing address.

39.

Click the Delete button to remove an existing address.

 

Step

Action

40.

The Rave system will generate a test message from no-reply at getrave.com to confirm your email address:

Dear FirstName,

This is the test message you requested. Case Western Reserve University is now able to reach you at this address.

 

Step

Action

41.

Click the Log Out link to exit.

42.

End of Procedure.
