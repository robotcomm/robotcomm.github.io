Windows VPN Issue

If you are experiencing issues connecting to VPN through Windows, please try the instructions below.

  1. Close the Cisco AnyConnect Window and the taskbar mini-icon
  2. Right click vpnui.exe in the “Cisco AnyConnect Secure Mobility Client” folder. (It may be in “C:\Program Files (x86)\Cisco\Cisco AnyConnect Secure Mobility Client\”
  3. Click on the “Run compatibility troubleshooter” button
  4. Choose “Try recommended settings”.
  5. The wizard suggests Windows 8 compatibility.
  6. Click “Test Program”.  This will open the program.
  7. Close the dialog window and connect to VPN normally.

Distributing the Evaluation

Questions must be added to a course evaluation at least seven calendar days before it is distributed to students. Questions included after this deadline will not be included in the course evaluation.

The system will deliver the course evaluation to students and issue periodic reminders. Course instructors are encouraged to reenforce the importance of course evaluations during class time.

Instructors and academic administrators (e.g., department chairs and deans) may run reports to view the results of course evaluations.

  • Instructions for course instructors
  • Instructions for academic administrators


Course Instructors

  1. Navigate to the course evaluations URL that was delivered to your CWRU email account.
  2. Go to the Results tab.
  3. Select Instructor Results.
  4. Check the box next to the appropriate project(s) from the project drop down menu (e.g., Spring 2015 Course Evaluations). Click Select.
  5. To view report for an individual course section, select View and choose the desired report.
  6. To generate a batch report, choose more than one course section and click Batch Report.
    1. Name the batch report.
    2. Select one of the following options and click Go:
      • Merge Multiple PDF Reports into one PDF for Selected Courses:
        Individual course sections reports will be compiled into a single PDF report—data is not aggregated
      • Download Multiple PDF Reports/Batch as ZIP File for Selected Courses: 
        Course section reports will be downloaded as individual PDF files into a ZIP file—data is not aggregated
      • Comparison Report with Selected Courses in one report and organized by survey questions as PDF:
        Comparison of each survey question from course to course—data is not aggregated

Academic Administrators

  1. Navigate to the course evaluations URL that was delivered to your CWRU email account.
  2. Go to the Results tab.
  3. Select Report Builder.
  4. Click Create New Report.
  5. Add report name and brief description. Click Continue.
  6. Check the box next to the appropriate project(s) (e.g., Spring 2015 Course Evaluations). Click Continue.
  7. Choose survey questions to add to report. Click Continue.
  8. Choose report “area” (e.g., respondent location in hierarchy; for example, limit to respondents from CAS Mathematics Department).
  9. Choose query parameters (e.g., question #1 where responses are selected and the response is greater than x_value).
  10. Click Save and then Continue.
  11. Choose a report template. Previews for each option are shown.
  12. Click Generate.
  13. Results are shown. NOTE: If no results match query parameters, No Data Found is displayed in red. 

Users are encouraged to seek assistance with this process by contacting the ITS Service Desk at 216.368.HELP (4357) or help@case.edu.
