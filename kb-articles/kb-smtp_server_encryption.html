SMTP Server 2048 Encryption

ITS uses 2048 bit encryption for the CWRU SMTP mail servers.

 

The following email clients are able to send mail through smtp.cwru.edu with authentication enabled:

  • Microsoft Outlook 2000 or greater
  • Microsoft Outlook Express version 5.0 or greater
  • Microsoft Entourage (all versions)
  • Microsoft Windows Mail 1.0 or greater
  • Mozilla Thunderbird 1.0 or greater
  • Qualcomm Eudora 6.2 or greater
  • Mail (Mac OS X)
  • Google Webmail

 

Individuals utilizing email client versions that do not support 2048 bit encryption must upgrade to one of the following mail clients:

  • Google Webmail online at webmail.case.edu
  • Microsoft Outlook 2010 available for free to students, faculty and staff at softwarecenter.case.edu
  • Mozilla Thunderbird version 3 or greater, available for free at www.mozilla.org/thunderbird
  • Apple Mail for OS X or above

 

2048 bit encryption also works with the following mobile devices:

  • Android OS version 2.0 or greater
  • iPhone/ iPad / iPod (all versions)
  • Blackberry 4.3 or greater
  • Palm Web OS 5.X or greater
  • Windows Mobile 5 or greater
  • Windows CE 4.0 or greater

Individuals utilizing mobile devices that do not support 2048 bit encryption should contact their service provider to determine the best possible upgrade path available.
