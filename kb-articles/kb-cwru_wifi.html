Connecting to Wireless Networks

Mac OS X Lion and Mountain Lion

  1. To begin, go to wireless-setup.case.edu in your internet browser.
  2. The Welcome page will appear. Click the link End-User License Agreement and review the agreement. When ready, click the check box and then click the Start button.
  3. A prompt will notify you that your Network ID and Password are required.
  4. Click OK.
  5. The instructions screen will appear. Follow the instructions listed on the screen to continue setup of your network profile for CaseWireless.

Windows 7 and Windows 8

Most Windows PCs include a switch or button that will enable wireless networking features. Some examples of these switches are shown in the figures below. If you cannot locate the switch for your particular PC model, you can consult the vendor’s support website for assistance.

  1. Log on to your Windows PC using your CWRU Network ID and password. If the wireless network is not activated on your
  2. Windows PC, the wireless icon in the system tray will appear as follows Locate the wireless switch/button and turn it ON.
  3. Using your internet browser, go to wireless-setup.case.edu.
  4. The Welcome page will appear. Click the link End-User License Agreement and review the agreement. When ready click the check box and then click the Start button.
  5. Follow the instructions listed on the screen to continue setup of the Network Wizard.
  6. If you have not installed the add-on for XpressConnect you will be prompted to install, click Yes to verify the publisher to continue.
  7. The Wizard will load, taking a few minutes; a pop up window will track the progress.
  8. The Network Wizard will open. Follow the instructions to finish set up of CaseWireless.
  9. If you receive a certificate message “You are about to install a certificate from a certification authority (CA) claiming to represent: InCommon Server CA…” please click Yes to continue.
  10. NOTE: You must be within range of the CaseWireless network to complete the set up with the wizard. If a connection error displays, make sure you are within range of the CaseWireless network.

iPhone

  1. From the iPhone home screen, tap the Safari button.
  2. Type the web address wireless-setup.case.edu.
  3. The Welcome page will appear. Click the link End-User License Agreement and review the agreement. When ready click the check box and then click the Start button.
  4. A prompt will notify you that your Network ID and Password are required.
  5. Tap OK.
  6. Tap the Install button to accept the Install Profile.
  7. Enter your CWRU Network ID in the username field and tap Next.
  8. Enter your Password in the Password field and tap Next.
  9. NOTE: If you have a lock on your phone you will be prompted to enter your passcode before this screen appears.
  10. The Profile will be installed. Tap Done.
  11. Upon completing the installation, follow the instructions on the screen to join the CaseWireless network.

About CWRU wireless networks

CaseWireless

Secure, authenticated network accessible on campus using a CWRU Network ID and password.

CaseWireless is a secure wireless network. Data sent and received over the network is protected from malicious threats by encryption technology. Users with a valid CWRU Network ID and password can access CaseWireless.

The CaseWireless network allows users to reach services that are specifically reserved for the CWRU community, such as the Software Center and Kelvin Smith Library databases.

Visit wireless-setup.case.edu to get started.

CaseGuest

Non-secure, public network accessible on campus without needing a username and password.

CaseGuest is a public wireless network. No username or password is required for access. Anyone within signal range can use it.

Because the CaseGuest network is designed to be unrestricted and accessible to all campus visitors, data processed on this network may be vulnerable to threat intrusion. Guests who need to access sensitive information are encouraged to secure their communications using a commercial Virtual Private Network (VPN) provider.

A Virtual Private Network (VPN) client is available at vpnsetup.case.edu for CWRU students, faculty and staff only. The Cisco® AnyConnect™ VPN client provides 128-bit Secure Socket Layer (SSL) encryption technology and is available for both Mac and Windows platforms.

EDUROAM (education roaming)

Secure, authenticated worldwide network accessible at participating institutions using a CWRU Network ID and password.

EDUROAM (eduroam.org) is a secure wireless network hosted by an international group of research and educational institutions for the purpose of providing students, faculty, staff and researchers with Internet connectivity when visiting participating institutions.

The eduroam network provides visitors from other universities with the ability to access a secure wireless network using their own log in credentials rather than having to obtain a local network ID and password. CWRU is a member of the eduroam organization and students, faculty and staff can access eduroam using their CWRU network ID and password whenever visiting an eduroam location.
