Google Docs

Google Docs is web-based word processing and spreadsheet program that makes collaboration more efficient. It lets you keep a document online that others can edit and update simultaneously right from their browser without the need to track attachments. Multiple people can make changes at once, and see edits from others as they happen in real time.

Google Docs can be accessed through these Single Sign-on sites:

  • webmail.case.edu 
  • webcalendar.case.edu 
  • webdocs.case.edu 
  • sites.case.edu
