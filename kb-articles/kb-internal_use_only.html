Internal Use Only Information

The types of information listed below are examples of information that is for internal university use only. If disclosed, it is considered to have a moderate level of impact upon the university. In general, all IT systems containing Internal Use Only information require user authentication using a valid CWRU Network ID for access to the informatoin. Tier I and Tier II Controls apply to systems with Internal Use Only Information. Note that CWRU Google Apps may host Internal Use Only information when protected by network authentication. Examples are:

  • university administrative information
  • strategic plans
  • internal only communications
  • home phone numbers
  • student directory information suppressed by FERPA
  • student grades
  • photos from the student directory
  • citizenship
  • visa status
  • redacted Social Security Number (last four digits only)
