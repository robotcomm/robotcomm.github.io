Accessing the System

If you are a course instructor, department chair or dean, a custom link to the new course evaluation system will be sent to your Case Western Reserve email account. Upon clicking the link, you will be prompted to sign in to EvaluationKIT using your CWRU Network ID and password.

Once logged in, you will see evaluations for each of the courses with which you are associated.

Distributing the Evaluation

The system will deliver the course evaluation to students and issue periodic reminders. Course instructors are encouraged to reenforce the importance of course evaluations during class time.

Instructors and academic administrators (e.g., department chairs and deans) may run reports to view the results of course evaluations. Before following these instructions, please visit the Understanding results reporting in EvaluationKIT page to learn more about how CWRU's new system compares to the system used in previous years. For additional assistance, or with help in creating more advanced reports, contact the ITS Service Desk at 216.368.HELP (4357).

Course Instructors

  1. Navigate to case.evaluationkit.com; sign in with your CWRU Network ID and password, if prompted.
  2. Go to the Results tab.
  3. Select Instructor Results.
  4. Check the box next to the appropriate project(s) from the project drop down menu (e.g., Undergraduate spring 2015 [93]). Click Select.
  5. You now have the option to create a report for each course that you instruct, or a "batch report" containing the evaluation responses from more than course that you instruct.

Individual Course Reporting

  1. To create a report for a specific course, click the View button adjacent to the desired course.‌
    ‌‌
  2. Choose desired format (e.g., PDF, PDF+Comments, Raw Data).
  3. The system will display a message confirming that the report is being generated.
  4. Choose to save the report to a location, or open with an application (e.g., Adobe Acrobat, Microsoft Excel).
  5. Open report using the appropriate application (e.g., Adobe Acrobat, Microsoft Excel).‌

Batch Course Reporting‌

  1. To report the results of more than one course together, click the selection box for each and then click Batch Report.‌
    ‌‌
  2. Name the batch report.
  3. Select one of the following options and click Go:

Merge Multiple PDF Reports into one PDF for Selected Courses:
Individual course sections reports will be compiled into a single PDF report—data is not aggregated

    Download Multiple PDF Reports/Batch as ZIP File for Selected Courses: 
    Course section reports will be downloaded as individual PDF files into a ZIP file—data is not aggregated

      Comparison Report with Selected Courses in one report and organized by survey questions as PDF:
      Comparison of each survey question from course to course—data is not aggregated

      The system will display a message confirming that the report is being generated and email the resulting file to you.

      Academic Administrators

      Academic administrators (e.g., department chairs and deans) may preview the results of one, some or all courses in their area.

      1. Navigate to case.evaluationkit.com; sign in using your CWRU Network ID and password, if prompted.
      2. Go to the Results tab; select Project Results.
      3. Click magifying glass in "View" column to open results.
      4. Choose to create a report of by hiearchy (e.g., department, school, college), Course Section or Instructor.

        ‌
        • Hiearchy: Select the school, college or department for which you want to gather results.

          Then, choose the desired format (e.g., PDF Report, PDF Report + Comments, Raw Data).

          ‌
        • By Course Section or By Instructor: Follow instructions for Indvidual Course Reporting or Batch Reporting in Instructor section above.
