Saving a Website Like an App

Use the directions below to save a website to your mobile device’s home screen so that it looks like an app.

iPhone:

 

  1. Use Safari to navigate to the website you want to add to your home screen.
  2. Tap the middle icon at the bottom of Safari (it looks like a box with an arrow pointing to the right).
  3. Tap “Add to Home Screen.”
  4. Enter a name for the icon if you want to change the default name.
  5. Tap the “Add” button. Safari will close and you will be taken to your new bookmark on the home screen.

 

 

Android:

 

  1. Use your web browser to navigate to the website you want to add to your home screen.
  2. Press the Menu button and select “Bookmarks.”
  3. Select the icon in the top-left that is labeled “Add.”
  4. Enter a name for the bookmark if you want to change the default name (note that this name will be what is displayed for the icon on the home screen)
  5. Press “OK” to add the bookmark.
  6. Press and hold on the newly-created bookmark and select “Add shortcut to Home” from the menu that pops up.

 

 

BlackBerry:

 

  1. Use your web browser to navigate to the website you want to add to your home screen.
  2. Press the BlackBerry button and select the “Add to Home Screen” option.
  3. Enter a name for the icon if you want to change the default name. You can also change where the icon will appear in the “Location” field, and you can add the icon to your Favorites panel by selecting the “Mark as Favorite” checkbox.
  4. Press the “Add” button to add the icon to the home screen.

 

 

Windows Phone 7:

 

  1. Use Internet Explorer to navigate to the website you want to add to your home screen.
  2. Tap the “…” icon in the bottom-right of the screen.
  3. Select the “Pin to Start” option to add the website to your home screen.
