Static IP Address

Certain machines need to have their IP Address remain static or unchanged for particular uses. These computers, typically servers, can obtain a Static IP Address upon request.

Note that Static IP Addresses are assigned by building, so if you request a static IP Address and then move to a different building, your computer will no longer work on the CWRU network. You will need to release and re-request the static IP Address. To request a static IP Address go to its-services.case.edu/NetworkTools/IPDB/staticRequestFrame.html.
