Install the Cisco AnyConnect VPN Client on a Windows Machine

  1. From the VPN installation webpage, click the Windows button.
  2. The file download window appears. Click the Save button.
  3. Select an appropriate location for the installation file to be saved and click Save.
  4. Return to your Desktop and locate the VPN Client installation file. Double-click the file.
  5. Click the OK button. The installation file will begin to open.
  6. The Setup Wizard appears. Click the Next button.
  7. Accept the terms of the End-User License Agreement (EULA).
  8. Click the Next button.
  9. Click the Install button.
  10. When the setup is complete, click the Finish button.
  11. Click Start (Windows button).
  12. Navigate All Programs > Cisco > Cisco AnyConnect VPN Client and execute program.
  13. The Cisco AnyConnect VPN Client appears. In the Connect to field, enter "vpn.case.edu".
  14. Click the Select and log with your Case Network ID and password.

About VPN at CWRU

Virtual Private Networking (VPN) software is used to remotely connect to the campus network, and yet appear to on-line resources as though you are directly connected. This software also helps secure your computer on the network so that traffic to and from your computer is encrypted.

The provider for VPN software at CWRU is Cisco AnyConnect. It is available at vpnsetup.case.edu.
