Types of Desk Phones

Model 7942G

 

This phone is the standard phone for full-time faculty and staff.

 

  • Two-line set
  • Hands free speakerphone
  • Large LCD display
  • Headset jack
  • Volume control
  • Message waiting light

Model 7945G

 
  • High-fidelity wideband audio for vibrant, lifelike conversations; Internet Low Bitrate Codec (iLBC) support for use in lossy networks
  • Large backlit color display for easy use of Cisco Unified Communications and third- party telephone applications
  • Improved navigation cluster for easier navigation and feature and function selection
  • Gigabit Ethernet connectivity

Model 7962G

 
  • Six-line set
  • Hands free speakerphone
  • Large LCD display
  • Headset jack
  • Volume control
  • Message waiting light

Model 7965G

 
  • High-fidelity wideband audio for vibrant, lifelike conversations; Internet Low Bitrate Codec (iLBC) support for use in lossy networks
  • Large backlit color display for easy use of Cisco Unified Communications and third-party telephone applications
  • Improved navigation cluster for easier navigation and feature and function selection
  • Gigabit Ethernet connectivity

Model 7937

 
  • Hands free speakerphone
  • Single-line set
  • Full duplex
  • Suitable for small to medium sized conference rooms
  • Optional microphone kit ($220.50) for large conference rooms
  • Model 7937 must be ordered from the vendor

Model 7970G

 
  • Eight-line set
  • Color touch-screen display
  • Hands free speakerphone
  • Headset jack
  • Volume control
  • Message waiting light

Model 7914 Expansion Module

 
  • Adds fourteen buttons for either additional lines or speed dial numbers
  • Includes new base and power cord
  • Up to two expansion modules can be added to a 7960
  • Model 7937 must be ordered from the vendor

Analog Telephone Adapter 186

 
  • Allows an analog and an IP phone to share the same faceplate

A - analog phone/fax
B - IP connection
C - power cord socket

Linksys EZXS55W Etherfast Switch

 
  • Linksys EZXS55W Etherfast 10/100 5 port auto- sensing switch includes power adapter.
  • The EtherFast 10/100 5-Port Switch is the easiest, most flexible way to boost your network's performance while migrating to the power of Fast Ethernet.
  • Designed to meet the needs of small businesses, medium businesses and SOHO networks.

Cisco Unified Wideband Handset

 
  • Enhanced audio experience makes communications more intelligible and comfortable for the user.
  • Sold as an accessory that can be installed on Cisco Unified IP Phone 7906G, 7911G, 7941G, 7942G, 7945G, 7941G-GE, 7961G, 7961G-GE, 7962G, 7965G, 7970G, and 7971G-GE, 7975G models.

Power Supply

 
  • Replacement power supply for most Cisco CP- 79XX series IP phones.
  • Compatible models include 7902G, 7910G+SW, CP-7940G 7905G, CP-7912G, CP-7960G 7910G, CP-7914.
