Service Description

The Technology Enhanced Classroom (TEC) initiative at CWRU enables instructors and students to access tools and resources available beyond the four walls of the traditional classroom. TECs provide the equipment and tools for a broader learning experience. From faculty lectures to student presentations, users are able to display multimedia and web-enabled information. The university supports more than 200 Technology Enhanced Classrooms.

Required Information

  • CWRU Network ID,
  • Contact information (preferred and alternate),

  • Urgency (e.g., class is in session)
  • Building and room number
  • TEC level

How to Escalate

Description and purpose

An escalation may be requested by an end user when he or she determines that a critical issue has not been acknowledged properly, remains unresolved for an unacceptable period of time or is severe enough that it requires atypical handling.

Escalation should not be confused with Forwardment to a resolver group. An escalated issue is typically brought to the attention of management within a second-level support group(s).

Who to contact for escalation

  1. Ray Tomorrowitz, (216) 368-5076 (office), (216) 832-2384 (mobile)
  2. Dwayne Bible, (216) 368-3246 (office), (216) 288-9503 (mobile)
  3. Michael Thomas, (216) 368-6232 (office), (216) 288-1081 (mobile)
  4. James Goodman, (216) 368-0774 (office), (216) 501-0697 (mobile)

Escalation procedures

TEC support is considered a critical service.

Escalations are available whenever requested, including during planned university closures and recognized university holidays.

During unplanned university closures escalations may NOT be available.

Classify the reason for escalation as one of the following; Confirmed Outage (total or partial), Suspected Outage (total or partial), Abend, Slow response time, User Requirements not met, Other functional or Other technical.

Use published schedule and contact information to route P0 and P1 incidents to the appropriate resolver group and current Ops Manager On Call.

Detailed Troubleshoot / Repair Procedures

Projector will not work laptop / Projector is “out of sync” / Projector displays a blue screen when pc is selected as source

return to index

  1. Determine whether immediate on-site assistance is needed.
  2. If yes, contact the MediaVision radio number, (216) 368-1622.
  3. MediaVision will dispatch a technician immediately.
  4. Technician does not answer the radio
  5. Wait for voicemail, press 0 for the On-Call Technician’s cell phone.
  6. Forward the ticket to the Technology Enhanced Classroom resolver group.
  7. If no, gather resolution information and enter it into the ticket.
  8. Advise the user to make sure the pc and projector are powered ON.
  9. Instructions for powering on TEC

  10. Determine the TEC type

    Type Brand of controller Size of controller ON/OFF operation Videoconferencing capable
    Level 1 SD (standard definition) Extron 5” X 7” Pushbutton No
    Level 2 SD (standard definition) AMX 5” X 7” Touch panel No
    Level 3 SD (standard definition) AMX 5” X 7” Touch panel Yes
    Level 1 HD (high definition) Crestron 10” X 14” Pushbutton No
    Level 2 HD (high definition) Crestron 10” X 14” Touch panel No
    Level 3 HD (high definition) Crestron 10” X 14” Touch panel Yes
    • Level 1 SD: Press the raised pushbutton on the controller unit to turn the system ON (green light should illuminate).
    • Level 2/3 SD: From the touch panel, press System Off in the upper - right corner and then press Yes. It will take 3 minutes for the system to power down and then 1 minute to power back up.
    • Level 1 HD: Press the raised pushbutton on the controller unit to turn the system ON (green light should illuminate).ight should illuminate.
    • Level 2/3 HD:
      1. From the touch panel, press Projector On/Off > Projector Off in top left corner.
      2. The user will be asked: Are you sure you want to turn off the projector?
      3. Select Yes, shut down. It will take 90 seconds for the projector to cool and then shut down.
      4. Press Projector On/Off > Projector On.
  11. PC users
  12. Toggle function F8 or F7 to ensure the external display is selected.
  13. Laptop resolution should be set to 1024 x 768 for a Standard Definition TEC, or 1280 x 720 for a High Definition TEC.
  14. MAC users
  15. From the Apple menu select System Preferences > Displays > Arrangement
  16. Select the Mirror Displays check box.
  17. If the Arrangement option is not visible, then the cable is not properly connected. Have the user double check the cable connection.
  18. Laptop resolution should be set to 1024 x 768 for a Standard Definition TEC, or 1280 x 720 for a High Definition TEC.
  19. If you are unable to assist the user, on-site assistance is needed. Contact a MediaVision technician via radio number (216) 368-1622.
  20. MediaVision will dispatch a technician immediately.
  21. Resolve the ticket or forward to the Technology Enhanced Classroom resolver group.

For TEC rooms that do not fall under the UGEN category, the user must contact the management center for the individual school to complete the room scheduling process.

Can I save files to the TEC computer?

return to index

  1. Gather resolution information and enter it into the ticket.

  2. For security purposes, the TEC computer has been protected with Faronics Deep Freeze software. All documents, settings or newly installed applications will be erased from the computer after it is logged out or restarted. When finished using the TEC computer, users must save files by one of the following methods to retain them:

    • Save to a USB flash drive
    • Upload to Google Docs/Drive
    • Send as email attachment(s) to their CWRU email address (or another address).

How do I run a PowerPoint presentation in a TEC from a flash drive?

return to index

  1. Gather resolution information and enter it into the ticket.
  2. Insert the Flash Drive into an open USB slot. The USB slot is located inside the grey face cover on the front of the computer.
  3. Touch the TOUCH TO START button (from the touch panel) to turn ON the navigation screen.
  4. Touch the right (next) arrow to proceed to the next screen.
  5. Touch the Room PC icon. At this time, the computer desktop should be displayed on the projector screen.
  6. Locate the PowerPoint presentation file by clicking the My Computer icon from the PC desktop and then navigating to the flash drive files.
  7. Double click the PowerPoint presentation file to open it.
  8. Resolve the ticket.

How do I run a PowerPoint presentation in a TEC from a laptop?

return to index

  1. Gather resolution information and enter it into the ticket.
  2. Hook up the monitor (VGA) cable to your laptop BEFORE TURNING ON THE LAPTOP.
  3. Turn the laptop ON.
  4. Touch the TOUCH TO START button (from the touch panel) to turn ON the navigation screen.
  5. Touch the right (next) arrow to proceed to the next screen.
  6. Touch the Laptop icon. At this time, the laptop desktop should be displayed on the projector screen.
  7. Locate the PowerPoint presentation files by clicking the My Computer icon from the PC desktop and then navigating to the folder where the user has the presentation stored.
  8. Double click the PowerPoint presentation file to open it.
  9. Resolve the ticket.

How do I use the DVD / VCR in a TEC?

return to index

  1. Gather resolution information and enter it into the ticket.
  2. Insert the DVD or VHS tape into the player.
  3. Touch the TOUCH TO START button (from the touch panel) to turn ON the navigation system.
  4. Touch the DVD/VCR icon to display the movie.
  5. Resolve the ticket.

How do I use an iPad with a laptop input?

return to index

  1. Gather resolution information and enter it into the ticket.
  2. Use an Apple adaptor to connect iPhone or iPad to the laptop cables.
  3. Select laptop from the touchscreen
  4. Level 2 HD
    1. Select VGA button when using VGA laptop cable
    2. Select HDMI button when using HDMI laptop cable
  5. If image does not appear after 15 seconds
    1. Leave the device connected and power down the iPad / iPhone
    2. Wait 15 seconds.
    3. Power on the device normally while the device remains connected to the laptop cable.
  6. If you are unable to assist the user, on-site assistance may be needed. Contact a MediaVision technician via radio number (216) 368-1622.
  7. MediaVision will dispatch a technician immediately.
  8. Technician does not answer the radio
  9. Wait for voicemail, press 0 for the On-Call Technician’s cell phone.
  10. Resolve the ticket or forward to Technology Enhanced Classroom resolver group.

NOTE: User must provide his or her own adaptor. MediaVision does not provide adaptors.
