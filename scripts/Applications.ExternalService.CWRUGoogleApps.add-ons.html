Escalation

Description and purpose

An escalation may be requested by an end user when he or she determines that a critical issue has not been acknowledged properly, remains unresolved for an unacceptable period of time or is severe enough that it requires atypical handling.

Escalation should not be confused with assignment to a resolver group. An escalated issue is typically brought to the attention of management within a second-level support group(s).

Detailed Activate / Setup / Install Procedures

return to top | index | setup questions

Install an add-on.

return to setup questions

  1. Gather resolution information and enter it into the ticket.
  2. Open and document or spreadsheet (using the new Sheets.)
  3. Click the Add-ons menu and then Get add-ons.
  4. Window appears showing all available add-ons.
  5. Select add-on and click the blue price button to install.
  6. For most add-ons, a message with appear requesting access to specific datat that the add-on needs to work. Read this message carefully and click Accept.
  7. Balloon notification appears when app is ready to use.
  8. Click the Add-ons menu to use the new add-on.
  9. Resolve the ticket.

Detailed Move / Change Procedures

return to top | index | change questions

Uninstall an add-on (from all files).

return to change questions

  1. Gather resolution information and enter it into the ticket.
  2. Open a document or spreadsheet (using the new Sheets).
  3. Click the Add-ons menu.
  4. Click Manage add-ons.
  5. Select add-on to remove from list.
  6. Click Manage.
  7. Select Remove.
  8. Resolve the ticket.

Turn on and turn off an add-on (for a specific file).

return to change questions

  1. Gather resolution information and enter it into the ticket.
  2. Open a document or spreadsheet (using the new Sheets).
  3. Click the Add-ons menu.
  4. Click Manage add-ons.
  5. From the list of add-ons, select the one you want to use.
  6. Click Manage.
  7. Click Use in this document. A checkmark will appear next to this option when the add-on has been turned on.
  8. To turn off the add-on for that file, click the option again.
  9. Resolve the ticket.
