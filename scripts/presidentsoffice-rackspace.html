Reset a Lost Password

back to top

  1. Verify user identity.
  2. Gather resolution information and enter it into the ticket.
  3. Assign ticket to DSS.

Reference Only

Admins can change mailbox details, including passwords, by editing the mailbox. Users should contact their office administrator to reset a password. Following to be performed by office administrator, DSS or escalation resolver ONLY.

To edit a mailbox, log into the Control Panel, and perform the following steps:

  1. Mouse over the Go to section drop-down menu and select Microsoft Exchange.
  2. In the Email Accounts section, click the Add / Edit Mailbox link.
  3. Click the Display Name you want to edit.
  4. Edit Password, taking care to meet the following password requirements:
    • Passwords must be at least 8 characters long.
    • Passwords must include at least 3 of the following: English uppercase characters; English lowercase characters; numerals (0-9); non-alphabet characters (such as !, #, $, %).
    • Avoid using characters from your Account Name (e.g., user@domain.com), Full Name, Initials, or Display Name.
    • The Requirements bar will be full when the password meets the minimum requirements.
  5. Retype the password to confirm.
  6. Click Save.
  7. Provide temporary password to user.
  8. User should then log into Control Panel and repeat steps above to change temporary password.

Set up a Hosted Exchange Mailbox on Sony Experia mobile and tablet devices using ActiveSync.

  1. Verify user identity.
  2. Gather resolution information and enter it into the ticket.
  3. Locate the Settings icon within your Apps Drawer.
  4. Within Settings, scroll down to Accounts and select Add account.
  5. Select Corporate.
  6. Enter the following information:
    • Email Address: first.last@case.edu
    • Password: Your mailbox password
  7. Tap Next.
  8. Enter the following information:
    • Domain\Username: first.last@case.edu
    • Password: Your mailbox password
    • Server: mex08.emailsrvr.com
    • Use secure connection (SSL): Checked
  9. Tap Next.
  10. If prompted with the Remove security administration alert, select Ok to proceed.
  11. Within the Account Settings screen, adjust the items you wish to sync.
  12. Resolve ticket.

Outlook (2007, 2010, 2013)

back to top

Set up Outlook with Hosted Exchange mailbox.

  1. Verify user identity.
  2. Gather resolution information and enter it into the ticket.
  3. Windows 7: Start Menu -> Control Panel. Double-click on Mail.
  4. Windows 8: Open the Charms by moving your mouse to the top right of your screen. Select Search. (Search for "Mail")
  5. When the Mail Control Panel opens, select Show Profiles.
  6. Select Add to add a new profile. Enter any name you wish and select OK.
  7. Select Manually configure server settings or additional server types. Click Next.
  8. On the Choose Service page, select Microsoft Exchange or compatible service. Click Next.
  9. On the Server Settings page, enter the following:
    • Server: outlook
    • Use Cached Exchange Mode: Checked
    • User Name: first.last@case.edu
  10. Click the More Settings... button.
  11. Go to the Connection tab.
  12. Under Outlook Anywhere, check "Connect to Microsoft Exchange using HTTP".
  13. Click the Exchange Proxy Settings... button.
  14. Within the box titled Use this URL to connect to my proxy server for Exchange, enter the following:
    • URL: "mex08.emailsrvr.com" (Without quotes)
    • Ensure both checkboxes are checked:
      1. Connect using SSL only
      2. Only connect to proxy servers that have this principal name in their certificate
  15. Within the box titled "Only connect to proxy servers that have this principal name in their certificate", enter the following:
    • "msstd:mex08.emailsrvr.com" (Without quotes)
    • Ensure both checkboxes are checked:
      • "On fast networks, connect using HTTP first, then connect using TCP/IP"
      • "On slow networks, connect using HTTP first, then connect using TCP/IP"
  16. Within the box titled Use this authentication when connecting to my proxy server for Exchange, select "Basic Authentication".
  17. Click OK in the Microsoft Exchange Proxy Settings window.
  18. Click OK in the Microsoft Exchange window.
  19. Within the window titled Add New Account, click Check Name". If you're prompted to enter your Username and Password, do so now.
  20. Once the Server" and User Name fields are underlined, click Next.
  21. Click Finish to complete the configuration.
  22. Resolve ticket.
