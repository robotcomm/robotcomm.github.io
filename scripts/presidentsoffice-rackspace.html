Sony

back to top

Set up a Hosted Exchange Mailbox on Sony Experia mobile and tablet devices using ActiveSync.

  1. Gather resolution information and enter it into the ticket.
  2. Locate the Settings icon within your Apps Drawer.
  3. Within Settings, scroll down to Accounts and select Add account.
  4. Select Corporate.
  5. Enter the following information:
    • Email Address: first.last@case.edu
    • Password: Your mailbox password
  6. Tap Next.
  7. Enter the following information:
    • Domain\Username: first.last@case.edu
    • Password: Your mailbox password
    • Server: mex08.emailsrvr.com
    • Use secure connection (SSL): Checked
  8. Tap Next.
  9. If prompted with the Remove security administration alert, select Ok to proceed.
  10. Within the Account Settings screen, adjust the items you wish to sync.
  11. Resolve ticket.

Set up an Apple iOS device with Hosted Exchange mailbox.

  1. Gather resolution information and enter it into the ticket.
  2. On the home screen, tap the Settings icon.
  3. Within Settings, tap Mail, Contacts, Calendars.
  4. Tap Add Account.
  5. Tap Exchange.
  6. Enter the following details:
    • Email Address: first.last@case.edu
    • Password: Your mailbox password
    • Description: Exchange Hosted Mailbox
  7. Tap Next.
  8. You may be prompted for additional information. If so, enter the following details:
    • Server: mex08.emailsrvr.com
    • Domain: Leave Blank
    • Username: first.last@case.edu
  9. Select the services that you wish to synchronize from the Exchange Server. Then tap Save.
  10. Resolve ticket.

Mac Mail

back to top

Set up Apple Mail with Hosted Exchange mailbox.

  1. Gather resolution information and enter it into the ticket.
  2. Launch System Preferences.
  3. Within System Preferences, click Internet Accounts.
  4. Then select Exchange.
  5. The pop up dialog will require the following:
    • Name: Enter your first and last name.
    • Email Address: first.last@case.edu
    • Password: Enter your mailbox password.
  6. Select Continue.
  7. If Server Address appears, enter "mex08.emailsrvr.com" (without quotes). Then click the Continue button once more.
  8. Select Continue. Then select the apps you would like to sync with Exchange.
  9. Select Done.
  10. Resolve ticket.

Outlook (2007, 2010, 2013)

back to top

Set up Outlook with Hosted Exchange mailbox.

  1. Gather resolution information and enter it into the ticket.
  2. Windows 7: Start Menu -> Control Panel. Double-click on Mail.
  3. Windows 8: Open the Charms by moving your mouse to the top right of your screen. Select Search. (Search for "Mail")
  4. When the Mail Control Panel opens, select Show Profiles.
  5. Select Add to add a new profile. Enter any name you wish and select OK.
  6. Select Manually configure server settings or additional server types. Click Next.
  7. On the Choose Service page, select Microsoft Exchange or compatible service. Click Next.
  8. On the Server Settings page, enter the following:
    • Server: outlook
    • Use Cached Exchange Mode: Checked
    • User Name: first.last@case.edu
  9. Click the More Settings... button.
  10. Go to the Connection tab.
  11. Under Outlook Anywhere, check "Connect to Microsoft Exchange using HTTP".
  12. Click the Exchange Proxy Settings... button.
  13. Within the box titled Use this URL to connect to my proxy server for Exchange, enter the following:
    • URL: "mex08.emailsrvr.com" (Without quotes)
    • Ensure both checkboxes are checked:
      1. Connect using SSL only
      2. Only connect to proxy servers that have this principal name in their certificate
  14. Within the box titled "Only connect to proxy servers that have this principal name in their certificate", enter the following:
    • "msstd:mex08.emailsrvr.com" (Without quotes)
    • Ensure both checkboxes are checked:
      • "On fast networks, connect using HTTP first, then connect using TCP/IP"
      • "On slow networks, connect using HTTP first, then connect using TCP/IP"
  15. Within the box titled Use this authentication when connecting to my proxy server for Exchange, select "Basic Authentication".
  16. Click OK in the Microsoft Exchange Proxy Settings window.
  17. Click OK in the Microsoft Exchange window.
  18. Within the window titled Add New Account, click Check Name". If you're prompted to enter your Username and Password, do so now.
  19. Once the Server" and User Name fields are underlined, click Next.
  20. Click Finish to complete the configuration.
  21. Resolve ticket.
