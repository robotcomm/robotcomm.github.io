Service Description

Weatherhead Career Link is a system provided by the Weatherhead Career Management Office that allows students to post resumes and search for jobs.

Student accounts are setup automatically. The end user does not need to request access Weatherhead Career Link. Access is granted on the basis of student status.

Authentication for Weatherhead Career Link is provided by Case Western Reserve Single Sign-On (SSO).

Tools for Resolution

Required Information

  • Name,
  • CWRU Network ID,
  • Contact information (preferred and alternate),
  • Problem or Request description

Resolver Groups

Weatherhead Helpdesk: Available Mon.-Fri., 8 a.m - 5 p.m., EST.

How to Escalate

Who to contact for escalation

Weatherhead Career Link is not considered a critical service. Escalations are not available at this time.

Description and purpose

An escalation may be requested by an end user when he or she determines that a critical issue has not been acknowledged properly, remains unresolved for an unacceptable period of time or is severe enough that it requires atypical handling.

Escalation should not be confused with assignment to a resolver group. An escalated issue is typically brought to the attention of management within a second-level support group(s).

Escalation procedures

Escalation is not available at this time. Assign all unresolved issues to the Weatherhead Helpdesk resolver group.

Troubleshoot + Repair

Student cannot log in to Career Link

return to top

  1. Gather resolution information and enter it into the ticket.
  2. Verify user has found the correct link as instructed below.
    1. Navigate to http://weatherhead.case.edu.
    2. Find the ‘Career Link’ image or link, and click to follow.
    3. The URL for the page that contains the link is http://weatherhead.case.edu/careers.
  3. If the user has found a Sign In screen with a URL similar to https://wsom-case-csm.symplicity.com/sso/students/, then Single Sign-On (SSO) has failed.
  4. Attempt to direct user to the correct URL as instructed above.
  5. If, after attempting to find the correct URL a second time SSO fails, instruct the student or alum to contact the Weatherhead Career Management office (CMO) directly.

  6. Weatherhead CMO contact information:

    • wsom-cmo@case.edu
    • (216) 368-3662

    NOTE: In most cases the Weatherhead CMO will need to add an account in the system for the user

  7. Resolve ticket.
