Service Description

Absolute Software Corporation (Absolute LoJack) provides persistent endpoint security and management for computers, laptops, tablets and smartphones. Absolute LoJack is available to ITS and College of Arts and Sciences (CAS) staff/faculty with manager pre-approval.

Use this article when:

  1. Any unlicensed user from CAS or ITS wishes to complete a manager approved install request (confirm with ticket number), or to create an install request for approval,
  2. Any licensed user requests to remove or change an existing installation.
  3. Any licensed user reports the theft or loss of a registered device.

Escalation should not be confused with assignment to a resolver group. An escalated issue is typically brought to the attention of management within a second-level support group(s).

How to install Absolute LoJack on a client machine

return to index

Download software and install

  1. Gather resolution information and enter it into the ticket.
  2. Connect to user device (client device).
  3. Navigate to cc.absolute.com
  4. Sign in with provided user ID and password (password is changed every 60 days, per vendor policy.)
  5. Click Helpful Links
  6. Download highest numbered version of package available for client device OS.
  7. Uncompress package on desktop of client machine. (.zip for Windows, .tar for Mac OS)
  8. Run Computrace (Windows) or RPclient (Mac OS) executable file.
  9. During package installation, do not change default settings.

Installation notes:
Mac OS – Untar RPclient.tar and install RPclient.pkg.
Windows – Double-click CompuTrace.exe to install.

Add client device to policy group

  1. After Absolute LoJack has been installed, run application CTMWeb (client device).
  2. When prompted, enter the password, “password”.
  3. Verify the ESN number on the status screen. If the last 4 digits are all zeroes the ESN value is invalid.
  4. Using the technician device, open the Computrace console.
  5. Click Admin Groups > Device Groups.
  6. Select appropriate group (CAS or ITS only).
  7. Click Add Devices.
  8. Search for client device by ESN.
  9. Select checkbox for correct client device.
  10. Click Choose Device to add client device to group.

CTM Web application notes:
Mac OS – Untar CTM.tar, then launch CTMWeb
Windows – Double-click CTMWeb.exe

Add additional device details to database record

  1. Type in client ESN and click Filter Members.
  2. Click the device identifier to open its record.
  3. Add Assigned Username: CASE Network ID, lowercase (i.e., rap141). Ignore ADS prefix if auto-detected by software.
  4. Add Email address in the following format: “first.last@case.edu”.
  5. Select applicable department.
  6. Click Save Changes.

Clean up client device

  1. Delete all compressed and uncompressed files from client device.
  2. Empty trash or recycle bin.
  3. Resolve the ticket.
