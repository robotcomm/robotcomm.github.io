Service Description

CWRU Connect (formerly Alumnet) is dedicated networking site for CWRU alumni. CWRU does not leverage LDAP for Single-Sign On (SSO), but instead relies on a separately maintained database. First time access to CWRU Connect is obtained through a two-step process.

A user must first register at CWRU Connect using their Full Name (first and last), home ZIP code and a Unique Identifier (UID) provided by CWRU (AdvanceID). After registering, a user must create a unique username and password. These credentials will be used to access the site after successful registration.

Identity verification

Verify user identity BEFORE you begin!

  • Contact information (preferred and alternate)

Resolver Groups

CWRU Connect: Available Mon.-Fri., 8 a.m - 5 p.m., EST. Inquiries placed before 5 p.m. will receive a same-day response. Inquiries placed outside of operating hours will receive a response on the next business day.

How to Escalate

Who to contact for escalation

  1. Brian Rosen, bdr2@case.edu, (216) 368-4404 (office)

Description and purpose

An escalation may be requested by an end user when he or she determines that a critical issue has not been acknowledged properly, remains unresolved for an unacceptable period of time or is severe enough that it requires atypical handling.

Escalation should not be confused with assignment to a resolver group. An escalated issue is typically brought to the attention of management within a second-level support group(s).

Escalation procedures

CWRU Connect is not considered a critical service.

Escalation is available Mon.-Fri., 8 a.m - 5 p.m., EST. Escalations are are not available during planned or unplanned university closures, or during recognized university holidays.

CWRU Connect is supported by the office of Advancement Services.

Classify the reason for escalation as one of the following; Confirmed Outage (total or partial), Suspected Outage (total or partial), Abend, Slow response time, User Requirements not met, Other functional or Other technical.

Troubleshoot + Repair

User has forgotten username or password.

back to top

  1. Gather resolution information and enter it into the ticket.

  2. Registered users

    1. Walk caller through password or username retrieval process.
    2. At login screen, instruct user to click retrieve password / retrieve username link and follow prompts.

  3. Unregistered users who have AdvanceID

    1. Walk user through process of registering at CWRUConnect.case.edu.
    2. Search for First and Last name.
    3. Enter Advance ID and ZIP.
    4. Follow registration prompts.
    5. Follow prompts to create username and password.
    6. Login with username and password.

  4. Unregistered users who DO NOT have AdvanceID

    1. Sign in to CWRU Connect ID Tool using your CWRU network ID and password.
    2. Verify user identity per resolution information instructions.
    3. Provide AdvanceID.
    4. Walk user through account registration.
    5. Walk user through username and password creation.

  5. Resolve the ticket or forward to CWRU Connect resolver group.

Activate + Setup + Install

User needs help registering for CWRU Connect.

back to top

  1. Gather resolution information and enter it into the ticket.
  2. Unregistered users who have AdvanceID

  3. Walk user through process of registering at CWRUConnect.case.edu.

    1. Search for First and Last name.
    2. Enter Advance ID and ZIP.
    3. Follow registration prompts.
    4. Follow prompts to create username and password.
    5. Login with username and password.

  4. Unregistered users who DO NOT have AdvanceID

    1. Sign in to CWRU Connect ID Tool using your CWRU network ID and password
    2. Verify user identity per resolution information instructions.
    3. Provide AdvanceID.
    4. Walk user through account registration.
    5. Walk user through username and password creation.
    6. Verify that user provided ZIP matches what is on file using the CWRU Connect ID Tool.
    7. Verify that user provided NAME matches what is on file using the CWRU Connect ID Tool.
    8. Note this and any other user information that is different from that found using CWRU Connect ID Tool in the ticket and forward to CWRU Connect resolver group.

  5. Verify the approximate cumulative number or credit hours completed at any CWRU school.

  6. If 12 or less, add to ticket and forward to CWRU Connect resolver group.
  7. Resolve the ticket or forward to CWRU Connect resolver group.—

ID Tool Instructions

back to top

  1. Sign in to awa.case.edu using YOUR (analyst) CWRU Network ID and password.
  2. Enter user (caller) ID or Name into search field.
  3. Press TAB (not enter) to reveal additional search options.
  4. Add more search criteria, or click Search to find possible record matches.
  5. Click entry in the Name column to add record to lookup.
  6. ID populated search box. Click full name to view complete record.
  7. Use information shown in Unique Identifiers form to verify identity.
  8. Click Home tab to execute new search.
