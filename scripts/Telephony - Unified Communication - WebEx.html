Cisco WebEx is comprised of four distinct applications and is part of Case Western Reserve University's unified communications suite of services.

Difficulty signing in

back to top

  1. Gather resolution information and enter it into the ticket.
  2. Verify that user has been granted access to WebEx by ITS. If NO, assign to RESOLVER - DAVE WEILACHER.
  3. Troubleshoot user wired or wireless connectivity, as appropriate.
  4. WebEx access is managed using SSO. Verify user is typing correct CWRU network ID and password.
  5. Sign in to cwru.webex.com using analyst CWRU network ID and password to verify service is available.
  6. Verify browser is supported. Note that Firefox is most broadly supported by manufacturer.
  7. Verify that user has followed appropriate setup steps, based on browser.
  8. Verify that Javascript and cookies are enabled (see, System Requirements).
  9. Optionally, you may instruct the user to try a test meeting at www.webex.com/test-meeting.html using Cisco's servers to verify they meet minimum system requirements.
  10. If analyst determines source of issue and it is not WebEx, consult appropriate script. Troubleshoot, assign, escalate or resolve as specified by subject matter instructions (e.g., Identity Management).
  11. Note the details of any troubleshooting steps taken, including the result.
  12. Assign unresolved tickets to RESOLVER - DAVE WEILACHER.

back to top

  1. Gather resolution information and enter it into the ticket.
  2. Verify that user has been granted access to WebEx by ITS. If NO, assign to RESOLVER - DAVE WEILACHER.
  3. Verify user has scheduled a meeting for requested date and time. If not, see Schedule meeting.
  4. Verify user is meeting host. If user is not host, see Difficulty attending a meeting.
  5. There are two simple ways to start a scheduled meeting, session or event:

    How Do I Start a Scheduled Meeting, Training Session, or Event?

    Either the host, or alternate host, of a meeting, session or event may start the online meeting at any time. To start your Meeting, Training session, or Event:

    From host confirmation email (preferred)

    1. Click provided link to start your WebEx meeting (browser will open cwru.webex.com).
    2. If the meeting does not automatically start, you may need to enter your CWRU network ID and password to log in. Click the Log In button.
    3. The meeting window will open.

    From cwru.webex.com (alternate)

    1. Sign in to cwru.webex.com with your Case network ID and password.
    2. Click the My WebEx button at the top of the page.
    3. On the My WebEx Meetings page, click Start (choose the link to the right of the meeting you would like to start.

    For information about starting or attending a meeting from the WebEx mobile application, see Troubleshoot mobile application.

  6. Sign in to cwru.webex.com using analyst CWRU network ID and password to verify service is available.

How-to + Training

Schedule a meeting, training session or event

back to top

  1. Gather resolution information and enter it into the ticket.
  2. Schedule a meeting using Meeting Center.

    1. Log in to cwru.webex.com using your CWRU network ID and password.
    2. Under Host a Meeting, click Schedule a Meeting.
    3. If you see the Advanced Scheduler page, click Quick Scheduler.
    4. Select a Meeting Type, enter a Meeting Topic, then enter and confirm the meeting password.
    5. Specify the Date, Time and Duration. Duration is for planning purposes only. The meeting will continue until you end it.
    6. Enter the email addresses of people you want to invite.
    7. Click the green Start button to begin the meeting now or click Schedule Meeting if you changed the time or date.
    8. To start a meeting you scheduled, click My Meetings, then locate the meeting and click Start.

  3. Schedule a training session using Training Center.

    1. Log in to cwru.webex.com using your CWRU network ID and password.
    2. Under Host a Session, click Schedule Training.
    3. Complete the Session and Access Information.
    4. Select the Audio Conference Settings.
    5. Specify the Date, Time and Duration. Duration is for planning purposes only. The meeting will continue until you end it.
    6. Check the boxes if you will require registration and if you want them approved automatically.
    7. Create invitation lists for Attendees and for Presenters. Enter new information for each person, or use existing contacts.
    8. Click Add Course Material if you want to make materials available for download before the session.
    9. Click Start Session to begin the session now or click Schedule to schedule the session and send invitations.
    10. To start a session you scheduled, under Attend a Session, click Live Sessions, and then locate the session and click Start.

  4. Schedule an event using Event Center.

    1. Log in to cwru.webex.com using your CWRU network ID and password.
    2. Under Host an Event, click Schedule an Event.
    3. Complete the Basic Information section.
    4. Enter a Date & Time, including any different time zones.
    5. Check the box if you want to allow people to join before the event's scheduled start time.
    6. Select the Audio Conference Settings.
    7. Use the Event Description & Options to customize the event. For example, you can upload material or choose whether to use video.
    8. Create invitation lists for Attendees and for Presenters & Panelists. Enter new information for each person, or use existing contacts.
    9. Choose which emails you want to send, and when to send them.
    10. Click Schedule This Event.
    11. When it's time to start the event, click Site Events, then locate your event and click Start.

  5. Resolve ticket.

Attend a meeting

back to top

  1. Gather resolution information and enter it into the ticket.
  2. Refer to end-user instructions below.

    If you are invited to attend a WebEx meeting, you will receive an email prior to the beginning of the meeting from the meeting organizer. This email contains important meeting details like the date and time of the meeting, call-in instructions and a link to join the meeting online.

    Many email clients and webmail applications, including CWRU webmail, allow you to create calendar events directly from email messages.

    Attend by telephone (audio only)

    1. Call conference number (toll-free) provided in invitation email.
    2. When prompted, enter meeting access code using telephone keypad.
    3. A tone will indicate that you have entered the meeting.

     

    Attend with computer

    1. Click the meeting link provided by the organizer (e.g., https://cwru.webex.com/join/CWRUnetID)
    2. If you have not previously attended a WebEx meeting, the system will prompt you to download and install the Meeting Center application.
    3. The WebEx application will launch and you will join the meeting room.
    4. Click Call Using Computer to connect meeting audio.
    5. Or, click the three dots below to show other audio options and directions:

      • Call Me (the meeting will call you)
      • I Will Call In (click to show instructions)
      • Call Using Computer (default)

    6. When audio has connected successfully, the icon will turn from blue to green and text will read "Connected to Audio".
    7. A list of meeting attendees as well as video & audio options are shown in upper-right of the application window.

     

    Attend with computer using telephone for audio

    1. Click the meeting link provided by the organizer (e.g., https://cwru.webex.com/join/CWRUnetID)
    2. If you have not previously attended a WebEx meeting, the system will prompt you to download and install the Meeting Center application.
    3. The WebEx application will launch and you will join the meeting room.
    4. A list of meeting attendees as well as video & audio options are shown in upper-right of the application window.
    5. Call conference number (toll-free) provided in invitation email.
    6. When prompted, enter meeting access code using telephone keypad.
    7. A tone will indicate that you have entered the meeting.

Use file transfer

back to top

  1. Gather resolution information and enter it into the ticket.

  4. By default, Meeting Center attendees have the ability to become the presenter in meetings by clicking and dragging the Presenter ball. Note the following limitations:

    Can I pass the Presenter Role to an attendee or participant?

    Anyone attending a WebEx session may view shared data, but only the person designated as the Presenter may share presentations, screen/desktop, or applications. The meeting or session host is normally the presenter in a meeting.

    Event Center
    Host may invite Panelists, who have presenter rights. Only the Host or Panelists may present in Event Center.
    Follow these steps to designate a presenter:
    1. On the Participants panel, select the name of the panelist whom you want to designate as the presenter.
    2. Do any of the following:
      From the Participants panel, click the Make presenter button
      From the Participant menu, choose Change Role to > Presenter
      Click and drag the Presenter ball to the desired Panelist
    Meeting Center
    Host can designate most meeting attendees as the presenter. Reclaim presenter role or change presenter at any time.
    Training Center
    Host may invite participants as Presenters, or pass the ball to any participant.

Use instant polling

back to top

  1. Gather resolution information and enter it into the ticket.
  2. Instant polling is only available to hosting using Meeting Center. Training Center and Event Center offer advanced polling. Support Center does not support polling.
  3. To open the polling panel click File > Open Poll Questions.
  4. Polling panel opens at right of application screen.
  5. Add or Edit Poll Questions Manually

    1. Click the icon with a question mark and "+" to add an instant poll question.
    2. If desired, click icon with a lightbulb and "+" to add answer to poll question.
    3. Repeat for all poll questions.
    4. To edit a question, select and click the pencil icon.
    5. To delete a question, select and click the trash icon.
    6. To change the order of questions, select question and click up or down arrow icon until question is in desired position.

  6. Add poll questions from a previously saved set

    1. Click the open folder icon at the bottom of the application screen.
    2. Use system finder to locate poll file (saved with .atp file extension).

  7. Save poll questions for reuse

    1. Click the floppy disk icon at the bottom of the application screen.
    2. Choose a file name and save location.
    3. Click Save As.

  8. Open poll for responses

    1. Choose whether to record individual responses.
    2. Click "Open Poll".
    3. Poll opens to meeting attendees.
    4. Time limit is set to 5 minutes, but poll may be closed manually before timer expires (click "Close Poll").
    5. By default, both percentage and count of attendees not started, in progress and finished are shown, as well as time remaining.

  9. Close poll, review and share results

    1. Allow timer to expire or click "Close Poll".
    2. Poll results are shown at upper-right of application screen.
    3. Select results to share with attendees (poll results, individual results, both or neither). If sharing with attendees, click "Apply" to save preferences. You may preview responses by clicking the link labelled "Individual results".

Use polling

back to top

  1. Gather resolution information and enter it into the ticket.
  2. How do I prepare a poll questionnaire?
    To conduct a poll, you must first create a poll questionnaire. You create a questionnaire in a meeting. To save time during a meeting, you can start the meeting earlier than the scheduled time, create a questionnaire on the Polling panel, save it, and then open it during the actual meeting.

    1. Open the Polling panel in a meeting.
    2. In the Question section, select one of these question types:
      • To create a multiple-answer question, select Multiple choice, and then select Multiple Answers in the drop-down list.
      • To create a single-answer question, select Multiple choice, and then select Single Answer in the drop-down list.
      • To create a text question, select Short answer.
    3. Click New.
    4. Type a question in the box that appears.
    5. In the Answer section, click Add.
    6. Type an answer in the box that appears.
    7. To type another answer, click Add once you finish typing an answer.
    8. Repeat steps to add more questions.

  3. Save and reopen a poll questionnaire

    Create polling questionnaire in a test meeting and save

    1. Open a test meeting.
    2. Create a poll questionnaire.
    3. Go to File, click on Save Poll Questionnaire.

Use breakout sessions

back to top

  1. Gather resolution information and enter it into the ticket.
  2. What is a breakout session?

    Breakout Sessions allow you to brainstorm and collaborate in small private groups away from the main training session. Participants can share information, chat, annotate, communicate via teleconference or VoIP, and pass the presenter ball. Once back in the main training session, information from breakout sessions can be shared with everyone in the main training session window.

  3. How do I create a breakout session for another participant?
    For host, panelists, or training session presenters only

    1. In the Breakout Session window, select the name of the participants whom you want to invite to the breakout session. To select more than one participant, hold down the Shift key while you select participants.
    2. Do either of the following:
      • On the Breakout menu, point to Breakout Session Assignment.
      • On the Breakout Session panel, click Breakout Assignment.
    3. Click Add Session (Windows) or + (Mac) to add a session.
    4. Name the session, or use the default name that is provided.
    5. Highlight the name of the breakout session.
    6. Highlight the names of any Not Assigned participant for whom you want to add to the breakout session and click the >> button.
    7. Select the radio button next to the participant that you would like to be the presenter.
    8. Click OK.

  4. How do I create a breakout session in the training session scheduler?

    1. Sign in to cwru.webex.com.
    2. On the navigation bar, expand Host a Session to view a list of links.
    3. Click Schedule Training.
    4. Scroll down the page to the Breakout Session Assignments Settings section.
    5. Check the Enable Pre-Session Assignment box, then select one of the following options:
      • Automatically assign attendees during the session
        Select, set the number of breakout sessions and enter a number into the text box (between 1 and 100)
        Select, set the number of attendees allowed and enter a number into the text box (between 1 and 100)
      • Manually assign registered attendees to breakout sessions.
        Note: You must enable registration to manually assign attendees to breakout sessions prior to your training session.

  5. Can I use voip during a breakout session?

    Voice over Internet Protocol (VoIP) is supported in Breakout Sessions. Start a VoIP conference from inside a Breakout Session:
    1. Go to the Audio menu.
    2. Select Integrated VoIP.
    3. Choose Start Conference.

  6. How do I start a breakout session?

    1. After your attendees have joined the meeting, go to the Breakout menu, then select Breakout Session Assignment....
    2. Chooose an assignment method:
      Automatically assign participants
      1. Select the Automatically radio button near the top of the window.
      2. Optionally, choose a the number of breakout sessions and attendees for each.
      3. Click OK.
      Manually assign participants
      1. Select the Manually radio button near the top of the window.
      2. Click Add Session and then name it.
      3. Click the name of the newly created Breakout session.
      4. With the Breakout Session highlighted, click the name(s) of the attendees you would like to add.
      5. Click the Assign button.
      6. When finished click OK.
      7. To start the Breakout Session, click the Start button on the Breakout Session panel.

Use hands-on labs

back to top

  1. Gather resolution information and enter it into the ticket.
  2. What is hands-on lab?

    Hands-on Lab sessions allow instructors to prepare lessons or exercises that students can complete on remote computers on which training software is installed. Students can use these remote computers during a training session for hands-on learning and practice. Because instructors can control Hands-on Lab sessions, they can maintain an optimized lab environment that is effective, familiar, and consistent.

  3. How do I setup a hands-on lab computer?

    1. Sign in to the Hands-On Lab Administration page from a computer you would like to add to the lab.
    2. Click Create New Lab.
    3. Name for the lab, add a description (optional) and click Add.
    4. On the Manage Labs page, click the name of the new lab.
    5. Click Set Up Computer.
    6. Follow on-screen instructions to complete the installation.
    Note: The Hand-on-Lab installation process via Firefox or Chrome is not available. As a workaround have the Admin use Internet Explorer browser to install the The Hand-on-Lab.

  4. Does hands-on lab support virtual machines?

    Virtual Machines are supported.

  5. How do I reserve computers for hands-on lab?

    1. Log in to cwru.webex.com using your CWRU network ID and password.
    2. Click the Training Center button near the top of the page.
    3. Click the Schedule Training link on the left-hand side of the page.
    4. Enter a topic and password for the session.
    5. In the Date and Time section, select the date and time for the session.
    6. Scroll down to the In-Session Hands-On Lab section.
    7. Select the Reserve computers from lab radio button.
    8. In the Reserve computers from lab drop-down menu, select the lab you would like to reserve computers from.
    9. In the Number of computers: field, select the desired number of computers to use from the lab.
    10. Optionally, click the Check Availability... button to confirm the lab will be available for the selected date and time.

  6. What are the limitations of hands-on lab?

    • Does not inherit attendee privileges from the Training Session.
    • Chat and annotation cannot be disabled.
    • Cannot assign an attendee to an already started breakout session.
    • Labs are only compatible with Microsoft Windows-based computers.
    • Web cam video is not supported.
    • Hands-on Lab computers are counted as a meeting participant.
