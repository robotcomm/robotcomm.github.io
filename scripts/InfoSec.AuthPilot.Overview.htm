Service Description

Multi-factor authentication (also, two-factor authentication, two-step verification, 2FA) provides additional layers of security to protect user data in the case of password theft. Duo Security two-factor authentication enhances the security of your CWRU accounts by using your phone or connected mobile device to verify your identity. This prevents anyone but you from accessing your CWRU accounts, even if they know your password.

NOTE: Authentication with Duo Security at CWRU is a pilot project. Full access to Duo Security is expected June 2014.

Use this article when:

  1. A pilot user reports a problem with a DuoSecurity mobile application, Duo Security enrollment or the CaseDuo VPN profile.
  2. A pilot user has a question about DuoSecurity enrollment, or using DuoSecurity with the CaseDuo VPN profile.
  3. Any user has a general question about multi-factor authentication or the CaseDuo pilot program.

NOTE: Any user may report a suspected outage of a DuoSecurity application or the CaseDuo VPN profile.

Detailed Activate / Setup / Install Procedures

return to top | index | setup questions

User requires assistance enrolling using a mobile phone.

return to setup questions

  1. Gather resolution information and enter it into the ticket.
  2. Step user through process below.
  3. Browse to the Duo Security self-enrollment page.
  4. Type in your CWRU network ID and password when prompted.
  5. Click or tap the green Start Setup button.
  6. Select the type of device you will use to enroll.
  7. Click or tap the green Continue button.
  8. Choose country code and enter phone number.
  9. Select the phone operating system. Choose Other if you are not enrolling a smartphone.
  10. Click or tap the green Continue button.
  11. Verify that the phone number is correct and click or tap the green Continue button.
  12. After successfully enrolling a device, choose “I’m done enrolling devices” to proceed or “Enroll another device”.
  13. Choose authenticator from enrolled devices, and authentication method (Duo Push, phone call, or passcode).
  14. Click or tap the green Log in button.
  15. Approve the authentication request (review the sms message, phone call, or Duo Mobile alert and enter the appropriate code into the second password field.)
  16. Resolve the ticket.

How do I use the multi-factor VPN profile?

return to training questions

  1. Gather resolution information and enter it into the ticket.
  2. First, enroll in DuoSecurity using a connected mobile device, cell phone or land line.
  3. Connect securely to CWRU network resources from a remote location using a VPN client.
  4. Choose the CaseDuo VPN profile from the Group dropdown menu.
  5. The user should enter his or her CWRU Network ID and password as normal.
  6. User will enter Second password provided by Duo Security authentication after enrollment.
  7. Click Connect or Ok to complete.
  8. Resolve the ticket.

What is multi-factor authentication?

return to training questions

  1. Gather resolution information and enter it into the ticket.
  2. Multi-factor authentication provides additional layers of security to protect user data in the case of password theft.
  3. Multi-factor authentication requires the user to present two or more of three authentication factors: a knowledge factor, a possession factor, and an inheritance factor.
  4. Resolve the ticket.

NOTE: Each factor must be validated by a service provider or third party before a user is granted access to the protected account..

What is a ‘knowledge factor’?

return to training questions

  1. Gather resolution information and enter it into the ticket.
  2. A “knowledge factor” is a user-created or system-assigned password.
  3. Provide details [CWRU password complexity requirements].
  4. Provide details [best practices for passwords].
  5. Provide details [physical security].
  6. Resolve the ticket.

What is a ‘possession factor’?

return to training questions

  1. Gather resolution information and enter it into the ticket.
  2. Token-code generators, text messages and digital certificate are common examples of the the “possession factor”.
  3. Provide details [physical security].
  4. Resolve the ticket.

What is an ‘inheritance’ factor?

return to training questions

  1. Gather resolution information and enter it into the ticket.
  2. “Inheritance factors” are not common in consumer applications.
  3. Most frequently used “inheritance factors” are biometric identifiers (e.g., fingerprint scanner or similar).
  4. Not used by CaseDUO.
  5. Resolve the ticket.
