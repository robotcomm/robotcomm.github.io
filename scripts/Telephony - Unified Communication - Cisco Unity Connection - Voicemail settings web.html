Service Description

Unified voice messaging at Case Western Reserve provides users with two paths to manage voicemail messages; a web-based interface and access via telephone. With the messaging assistant, users are able to,

  • Record voice messages using a computer
  • Set personal preferences for message playback (e.g., speed, order, etc.)
  • Set or change a voicemail passcode (PIN)
  • Record and set voicemail greetings

Description and purpose

An escalation may be requested by an end user when he or she determines that a critical issue has not been acknowledged properly, remains unresolved for an unacceptable period of time or is severe enough that it requires atypical handling.

Escalation should not be confused with assignment to a resolver group. An escalated issue is typically brought to the attention of management within a second-level support group(s).

Escalation procedures

Escalations are available whenever requested, including during planned university closures and recognized university holidays.

During unplanned university closures escalations may NOT be available.

Classify the reason for escalation as one of the following; Confirmed Outage (total or partial), Suspected Outage (total or partial), Abend, Slow response time, User Requirements not met, Other functional or Other technical.

How do I change or record my name

back to top

  1. Gather resolution information and enter it into the ticket.
  2. Go to unity-vm1.case.edu/inbox/.
  3. Enter CWRU network ID and password when prompted
  4. Choose Settings to open the “Messaging Assistant”.

  5. Choose menu options Preferences > Personal**

    Add an alternate spelling of first or last name

  6. Type alternate spellings in labelled boxes.
  7. Click the Save button at the BOTTOM of the page.

  8. Confirmation message appears, “Your changes were successfully saved”.

    Add an alternate name

  9. Select “Add Row” under the boxes labelled “Alternate Names”.
  10. Enter name in the first and last name boxes.
  11. Repeat for each additional alternate name.
  12. Click the Save button at the BOTTOM of the page.

  13. Confirmation message appears, “Your changes were successfully saved”.

    Remove an alternate name(s)

  14. Select checkbox next alternate name(s) to remove. NOTE: Select all names with the “Select All” control. Clear all selections with “Clear All”.
  15. Click “Delete Selected” to remove selected name(s).
  16. Click the Save button at the BOTTOM of the page.

  17. Confirmation message appears, “Your changes were successfully saved”.

    Play or record name

  18. Click “Play/Record” NOTE: this option requires Java browser plugins. The current version (v8 update 25) is available from Oracle.

  19. Leave the default playback and record options for best results (record with phone).

    Record or re-record name

  20. Click the record button (represented by a red circle).
  21. System will call your default telephone number and prompt recording.
  22. Press pause button (represented by two vertical bars) to pause (and resume) recording.

  23. Press the stop button (represented by a black square) to stop recording.

    Playback recorded name

  24. Click the play button (represented by a triangle pointing right) to hear recorded name.
  25. Resolve ticket.

How do I set a play order for new messages

back to top

  3. Enter CWRU network ID and password when prompted.
  4. Choose Settings to open the “Messaging Assistant”.
  5. Choose menu options Preferences > Message Playback.
  6. Arrange message attributes in preferred priority order.
  7. Select an attribute and use the “Move Up” and “Move Down” controls to change its position in order (default order shown):
    • Urgent voice messages
    • Normal voice messages
    • Urgent faxes
    • Normal faxes
    • Urgent emails
    • Normal emails
    • Receipts and notices
  8. Choose to play newest (default) or oldest messages first.
  9. Click the Save button at the BOTTOM of the page.
  10. Confirmation message appears, “Your changes were successfully saved”.
  11. Resolve ticket.

What greeting options are available

back to top

  1. Gather resolution information and enter it into the ticket.
  2. The following greeting types are available.
    • Alternate (disabled by default - when enabled, overrides all other greetings)
    • Busy (disabled by default)
    • Internal (disabled by default)
    • Closed (disabled by default)
    • Standard (enabled default)
    • Holiday (disabled by default)
  3. When enabled, each greeting type may use a system default greeting, personal recording or nothing at all.
  4. Only one greeting type may be enabled at a time.
  5. Resolve ticket.

How do I enable a greeting

back to top

  1. Gather resolution information and enter it into the ticket.
  2. Go to unity-vm1.case.edu/inbox/.
  3. Enter CWRU network ID and password when prompted.
  4. Choose Settings to open the “Messaging Assistant”.
  5. Choose menu options Greetings > View All Greetings.
    1 Click the greeting name (in the column marked “Greeting”).
  6. Choose Status option (choose one)
    • Disabled
    • Enabled with no end date and time
    • Enabled until …
      1. Choose end date (Month, Day, Year)
      2. Choose end time (HH:MM AM/PM)
  7. Choose Greeting Audio option (choose one)
    • My Personal Recording (record, re-record or review recorded greeting, if needed)
    • System default greeting
    • Nothing
  8. If using a personal recording, also select Callers See, “My Personal Recording”.
  9. Click the Save button at the BOTTOM of the page.
  10. Confirmation message appears, “Your changes were successfully saved”.
  11. Resolve ticket.

How do I view a personal contact

back to top

  1. Gather resolution information and enter it into the ticket.
  2. Go to unity-vm1.case.edu/inbox/.
  3. Enter CWRU network ID and password when prompted.
  4. Choose Settings to open the “Messaging Assistant”.
  5. Choose menu options Preferences > View Contacts.
  6. Any personal contacts entered will be shown in a grid.
  7. This grid is blank by default (campus directory is populated by LDAP).
  8. The following information is shown for each contact,
    • First Name
    • Last Name
    • email address
    • Dialed work, home and mobile numbers (used to call contact by voice command)
    • Work, home and mobile numbers (used for personal call transfer rules)
  9. Users may provide alternate spellings of first and last name in the record detail screen (click contact First Name to view or edit)
  10. Resolve ticket.
