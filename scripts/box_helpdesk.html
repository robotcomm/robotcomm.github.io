How to Escalate

Description and purpose

An escalation may be requested by an end user when he or she determines that a critical issue has not been acknowledged properly, remains unresolved for an unacceptable period of time or is severe enough that it requires atypical handling.

Escalation should not be confused with assignment to a resolver group. An escalated issue is typically brought to the attention of management within a second-level support group(s).

Troubleshoot + Repair

SSO, Username, Password issues

Because Box.com access is integrated with single sign-on, account access issues are best addressed by the Identity Management resolver group.

  1. Gather resolution information and enter it into the ticket.
  2. Troubleshoot CWRU Network ID and password issues, as appropriate.
  3. Troubleshoot connectivity issues, as appropriate.
  4. Record any troubleshooting steps taken in the ticket.
  5. Record any error messages received.
  6. Resolve the ticket or forward to Identity Management resolver group.

User receives ‘File In Transition’ message

If, when trying to download a file, you encounter an error message that includes the message, ‘File in Transition,’ please contact the ITS Box resolver group for further assistance.

  1. Gather resolution information and enter it into the ticket.
  2. Record name of the file user is trying to download.
  3. Record location of the file and time the file was uploaded.
  4. Forward to ITS Box resolver group.

User page is not displaying properly

  1. Gather resolution information and enter it into the ticket.
  2. Ensure that user web browser is either the latest version of Internet Explorer, Safari or Firefox.
  3. If user is able to see the ‘View: List’ or ‘View: Icon’ feature on the ‘All Files’ page, try toggling between the two views by clicking on them.
  4. Clear user’s browser cache.
  5. Verify that Box.com is recognized by installed security software as a trusted site.
  6. Instruct user to attempt access to Box.com from a different computer.
  7. Record troubleshooting steps taken in ticket.
  8. Resolve the ticket or forward to school-specific resolver group (see chart of non-ITS resolver groups).

User receives ‘ZIP file is corrupted’ message.

When a batch of files are set to download to your desktop as a zip file, the zip will appear on your desktop before the download is completed. If you attempt to open the file before the download is complete, an error message will appear.

  1. Gather resolution information and enter it into the ticket.
  2. Advise user, please allow the zip file to finish its download before attempting to open it.
  3. Resolve the ticket.

Troubleshooting FTP connections

  1. Gather resolution information and enter it into the ticket.
  2. Refer to the article below.
  3. Resolve the ticket or forward to school-specific resolver group (see chart of non-ITS resolver groups).

Box supports FTP (file transfer protocol) for Business and Enterprise customers. The purpose of FTP is for bulk migration of data. FTP is designed to be used for initial bulk upload and occasional bulk download of files from your account – we do not recommend FTP as your main access method.

The user can either use FTP, FTPS, or FTPES. SFTP is not supported. Note that when you upload a file from ftp, you will not get email notifications.

To Connect to Box via Command Line

  1. FTP:

    1. Open a terminal session or command prompt. Type ftp ftp.box.com. Then enter your username and password.
    2. Type passive
    3. Type epsv
    4. Use the lcd command to change your local directory
    5. For example: lcd /Users/user/Desktop
    6. To upload one file use put
    7. ftp> put myfile.pdf
    8. To upload many files use mput
    9. ftp> mput *.htm
    10. Type quit to close the connection and go back to your original terminal window.

  2. FTPS:

    • Connect to Box via ftps over Port 21 (explicit mode, aka FTPES):
      shell> curl -3 -v --disable-epsv --ftp-skip-pasv-ip -u username@example.com --ftp-ssl ftp://ftp.box.com

    • Connect to Box via ftps over Port 990 (implicit mode, no handshake):
      shell> curl -3 -v --disable-epsv --ftp-skip-pasv-ip -u username@example.com --ftp-ssl ftps://ftp.box.com:990

    • Upload a file:
      shell> curl -3 -v --disable-epsv --ftp-skip-pasv-ip -u username@example.com --ftp-ssl --upload-file filename ftp://ftp.box.com

SSO (Single Sign On)

FTP does not support SSO login credentials.

  1. Create a Box specific password to supplement your SSO login.
  2. Visit app.box.com/settings and select Create External Password.
  3. This password can be used for external applications that do not support SSO.

Two-Step Verification

If you have Two-step verification turned on for your account the workflow for logging in is as follows:

  1. Try and login normally with your Box specific password
  2. You will receive a text message containing a 16 digit application password
  3. Login again with that 16 digit code as your actual password

Sorting by file name

  1. Gather resolution information and enter it into the ticket.
  2. To view files in numerical order when titles with numbers, simply choose ‘View by’ then select ‘Name’.
  3. For numbering systems that are double digit or higher “0” must be used to fill digit spaces for the numbers to display correctly. For example, if you have a list of items that goes to 15, use “01” instead of “1” for the order to display correctly.
  4. Resolve the ticket

New subfolders not syncing to desktop

  1. Gather resolution information and enter it into the ticket.
  2. Content on Box.com but not appearing locally is a result of content not being marked fully for Sync on Box.com.
  3. When a parent folder is partially synced (displays the Some Items Synced status in the More Options dropdown), new subfolders created within this parent folder will not be marked for Sync automatically.
  4. To fully sync a partially synced folder, select Sync all items in this folder.
  5. Resolve the ticket

Install Box Sync

  1. Gather resolution information and enter it into the ticket.
  2. Refer to the article below.
  3. Resolve the ticket.

Windows:

  1. Download the Box Sync Setup EXE file.
  2. Open the EXE file from the Windows folder and follow the installation instructions.
  3. Box Sync will start automatically once the installation is complete.
  4. Your synced files are stored in a new folder called Box Sync.

Access Box Sync folder from the Favorites section in the left pane of Windows Explorer. Or, right-click the Box Sync icon in the system tray and click Open Sync Folder.

  1. Download the Box Sync Installer DMG file from box.com/sync4mac
  2. Open the DMG file from the Mac folder.
  3. Double-click on Box Sync.
  4. Your synced files are stored in a new folder called Box Sync.

Access Box Sync folder from the Favorites section in the left pane of your Finder window.

Uninstall Box Sync

  1. Gather resolution information and enter it into the ticket.
  2. Quit the Box Sync application form the task bar menu.
  3. Refer to the article below.
  4. Resolve the ticket.

Windows:

  1. Uninstall Box Sync from the Control Panel.

  2. Delete the following folders (note that AppData is hidden by default):

    C:\Users\[username]\AppData\Local\Box\Box Sync\
    C:\Users\[username]\Box Sync\

Mac:

  1. Delete Box Sync from your Applications folder.

  2. Delete the following folders (note that Library is hidden by default):

    Macintosh HD/Users/[username]/Library/Logs/Box/Box Sync/
    Macintosh HD/Users/[username]/Box Sync/
    Macintosh HD/Users/[username]/Library/Application Support/Box/Box Sync/

How-to + Training

Create new files and folders.

  1. Gather resolution information and enter it into the ticket.
  2. Right-click in the Files and Folders screen or select one of the options from the dropdown menu revealed when you click the ‘New’ button in the top-left corner.
  3. Refer to instructions for folders, Google Docs and Microsoft Office below.
  4. Resolve the ticket.

Creating Folders

  1. Click the New button or right-click in the Files and Folders screen
  2. Select New Folder from the dropdown menu
  3. Enter a name in the Folder Name field
  4. Select to either (1) keep the folder private or (2) invite collaborators.
  5. To invite collaborators, enter their names or email addresses in the entry field and select access level
  6. Click Okay

Creating Google Docs and Spreadsheets

Create and share Google documents directly from your Box account.

  1. Click the New button in the top-right corner of the Files and Folders screen
  2. Select New Google Doc or New Google Spreadsheet from the dropdown menu
  3. Name your Google document and click Okay
  4. Edit your Google document
  5. Close file when finished; changes are automatically saved

Office Docs

Create documents with Microsoft Office directly from your Box account.

  1. Click the New button in the top-right corner of the Files and Folders screen
  2. Select New Word Document, New PowerPoint Document, or New Excel Spreadsheet from the dropdown menu
  3. Name your Office document
  4. Edit your document in Microsoft Office
  5. Close file when finished; changes are automatically saved

Lock files while editing (version control).

  1. Gather resolution information and enter it into the ticket.
  2. Prevent save conflicts when working collaboratively on a file by locking it.
  3. Choose whether others can download the file while locked.
  4. Set a lock expiration time so that you do not need to unlock manually.
  5. When finished, click ‘unlock’ and other users will have the ability to edit the file (any other collaborator with edit access may unlock the file if an expiration time has not been set).
  6. To concurrently edit a document, use a Box Note or Google Doc from Box, not a Microsoft Office document.
  7. Resolve the ticket.

Rename a file or folder

  1. Gather resolution information and enter it into the ticket.
  2. Mouse over the filename
  3. Click on the “pencil” icon
  4. Rename the file

Edit an existing Box Note.

  1. Gather resolution information and enter it into the ticket.
  2. Click the filename to open a Box Note in a new tab or window in your browser.
  3. To change the way text is formatted in Box Notes, use the toolbar at the top. Select from the following specific formatting tools:
    • Font Size: Select the size (small, medium, or large) from the dropdown list.
    • Font Color: Select the required color block from the dropdown list.
    • Font Style: Apply bold, italic, or underline styles.
    • Bold: Click the B icon to apply the bold style or use the Command and b keyboard shortcut.
    • Italic: Click the I icon to apply the italic style or use the Command and i keyboard shortcut.
    • Underline: Click the U icon to apply the underline style or use the Command and u keyboard shortcut.
    • Text Alignment: Select one of these options to align your text to the left, center, or right of the note.
    • Numbered or Bulleted Lists: Click the numbered icon to add a numbered list or the bulleted icon to add a bulleted list to your note. Use the Tab key to further indent either list.

Access levels for collaborators.

  1. Gather resolution information and enter it into the ticket.
  2. Refer to the article below.
  3. Resolve the ticket.

When inviting a user as a collaborator within a folder, you will have the ability to set the level of access that user has to your content. Refer to the following chart to review the different permissions that accompany each access level:

Previewer
Uploader
Previewer-Uploader
Viewer
Viewer-Uploader
Editor
Co-Owner



Access Details

Viewer (read access): Can preview, download, make comments, and generate shared links. They will not be able to add tags, invite new collaborators, edit shared links, upload, edit or delete items in the folder.

Editor (read/write access): Can view, download, upload, edit, delete, copy, move, rename, generate and edit shared links, make comments, assign tasks, create tags, and invite/remove collaborators. They will not be able to delete or move root level folders.

Co-Owner (read/write access + manage users): Can add new collaborators, change access levels of existing collaborators, and remove collaborators. However, they will not be able to manipulate the Owner of the folder or transfer ownership to another user.


Change the owner of a folder.

  1. Gather resolution information and enter it into the ticket.
  2. From within your Box account, navigate to the folder you would like to transfer
  3. Click into the folder to see the list of active collaborators in the side menu
  4. To select another collaborator as the owner, hover your mouse over their name and a small pop-up window will appear
  5. Select Folder Owner from the drop-down of available choices
  6. Confirm your selection when prompted and click Okay to complete the process
  7. Once completed, the collaborator you designate will be the new Owner of the folder and your access level will be downgraded to Editor.
  8. The contents of folder will no longer count against your account’s storage allowance.
  9. Resolve the ticket.

Manage tasks workflow.

NOTE: Currently, Tasks can only be created if comments are enabled.

  1. Gather resolution information and enter it into the ticket.
  2. Refer to the brief article below.
  3. Resolve the ticket.

Create a task

  1. In the comments section of the file page, select ‘assign a task.’
  2. The comments section will convert to an ‘Add Task’ section.
  3. Add a brief description in the main text field.
  4. Select a task type by clicking the yellow icon under the description section.
  5. Enter the name of the collaborator to whom you will assign the task.
  6. Click ‘Assign Task’ to complete and assign your task.

Edit a Task

  1. Only the creator of the task can edit the task.
  2. Locate the file associated with the task
  3. Click the comment icon to the right of the file name.
  4. Mouse over the task, then click on ‘Edit’ to make changes to the task.

Complete a Task

  1. Click the task link provided in email notification
  2. To complete the task, select the Complete option on the task and click Yes to confirm when prompted.
  3. Leave a comment on the task without completing it, by mousing over the task and clicking Reply.
