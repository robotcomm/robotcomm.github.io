Service Description

Unified voice messaging at Case Western Reserve provides users a web-based interface to manage telephone settings. Using Phonesetup.case.edu users are able to,

  • Add, change and delete speed dial entries
  • Change voicemail notification settings
  • Enable or disable on-phone call history
  • Set call forwarding rules
  • Enable or disable “Do Not Disturb” mode

Description and purpose

An escalation may be requested by an end user when he or she determines that a critical issue has not been acknowledged properly, remains unresolved for an unacceptable period of time or is severe enough that it requires atypical handling.

Escalation should not be confused with assignment to a resolver group. An escalated issue is typically brought to the attention of management within a second-level support group(s).

Escalation procedures

Escalations are available whenever requested, including during planned university closures and recognized university holidays.

During unplanned university closures escalations may NOT be available.

Classify the reason for escalation as one of the following; Confirmed Outage (total or partial), Suspected Outage (total or partial), Abend, Slow response time, User Requirements not met, Other functional or Other technical.

Speed dial options

back to top

  1. Gather resolution information and enter it into the ticket.
  2. Determine whether user is calling about a shared mailbox or in individual mailbox.
  3. If shared mailbox, STOP. These instructions are for individual mailboxes ONLY. Return to knowledgebase and search for appropriate instructions.
  4. Go to phonesetup.case.edu.
  5. Enter CWRU network ID and password when prompted.
  6. Select “Phone Settings” from the left side navigation. NOTE: Menu item is highlighted in blue when selected.

  7. Click the “Speed Dial Numbers” label to expand it,

    NOTE: to select different options for telephones assigned the same number, first click the “link” button to unlink equipment profiles.

    Add a speed dial

    1. Click Add New Speed Dial

    2. In the pop-up window specify,

      • Number/URI
      • Label (Description)
      • Speed dial number

      NOTE: all three items are required to proceed.

    3. Number/URI may be a telephone number, Jabber client ID or Telepresence endpoint number (e.g., conferenceroom1@case.edu).
    4. Click Ok to continue or Cancel to abandon.
    5. Addition is successful when new speed dial information is displayed in “Phone Settings” screen.

    Edit a speed dial

    1. Click the “pencil” icon to edit the corresponding speed dial entry.
    2. Click Yes to confirm deletion or No to abandon.
    3. Edit is successful when new speed dial information is displayed in “Phone Settings” screen.

    Delete a speed dial

    1. Click the “X” icon to delete the corresponding speed dial entry.
    2. Click Ok to continue or Cancel to abandon.
    3. Deletion is successful selected speed dial information is no longer displayed in “Phone Settings” screen.

  8. Click Save to confirm all changes or Cancel to abandon.

  9. Resolve ticket.

back to top

  6. Select “Phone Settings” from the left side navigation. NOTE: Menu item is highlighted in blue when selected.

  8. Select or deselect any of the following options for each phone number that is managed under your account,

How do I set call forwarding rules

back to top

  1. Gather resolution information and enter it into the ticket.
  2. Determine whether user is calling about a shared mailbox or in individual mailbox.
  3. If shared mailbox, STOP. These instructions are for individual mailboxes ONLY. Return to knowledgebase and search for appropriate instructions.
  4. Go to phonesetup.case.edu.
  5. Enter CWRU network ID and password when prompted.
  6. Select “Call Forwarding” from the left side navigation. NOTE: Menu item is highlighted in blue when selected.

  7. Click the your telephone number label to expand it,

    Apply call forwarding to all incoming calls

    1. Click the checkbox next to Forward all calls to: option to select it.
    2. Choose correct call forwarding number from dropdown menu to the right (default is Voicemail).
    3. If correct number is not shown, erase the contents of the box, and type new forwarding number.
    4. Click Save to proceed.
    5. Settings have been saved correctly when green confirmation message appears.

    Advanced calling rules
    Using advanced calling rules, users may specify (for each line) forwarding destinations for each of the following scenarios separately,

    • Internal calls, line is busy
    • Internal calls, no answer
    • External calls, line is busy
    • External calls, no answer

    NOTE Advanced calling rules may not be applied while Forward all calls to: option is selected.

  8. Resolve ticket.

How do I enable or disable “Do Not Disturb”

back to top

  1. Gather resolution information and enter it into the ticket.
  2. Determine whether user is calling about a shared mailbox or in individual mailbox.
  3. If shared mailbox, STOP. These instructions are for individual mailboxes ONLY. Return to knowledgebase and search for appropriate instructions.
  4. Go to phonesetup.case.edu.
  5. Enter CWRU network ID and password when prompted.
  6. Select the “IM & Availability” tab at the top of the page. NOTE: Tab appears recessed when selected.
  7. In the form section marked “Do Not Disturb” select the option, Turn on.
  8. Deselect the Turn On option to turn off.
  9. Click Save to confirm all changes or Cancel to abandon.
  10. Settings have been saved correctly when green confirmation message appears.
  11. Resolve ticket.
