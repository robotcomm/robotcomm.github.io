Service Description

Unified voice messaging at Case Western Reserve provides users with two paths to manage voicemail messages; a web-based interface and access via telephone. Users are able to,

  • Record voice messages using a computer
  • Manage voice messages in a familiar, email-like environment

How to escalate

Who to contact

  1. Telephone Services

Description and purpose

An escalation may be requested by an end user when he or she determines that a critical issue has not been acknowledged properly, remains unresolved for an unacceptable period of time or is severe enough that it requires atypical handling.

Escalation should not be confused with assignment to a resolver group. An escalated issue is typically brought to the attention of management within a second-level support group(s).

Escalation procedures

Escalations are available whenever requested, including during planned university closures and recognized university holidays.

During unplanned university closures escalations may NOT be available.

Classify the reason for escalation as one of the following; Confirmed Outage (total or partial), Suspected Outage (total or partial), Abend, Slow response time, User Requirements not met, Other functional or Other technical.

How do I compose new messages

back to top

  1. Gather resolution information and enter it into the ticket.
  2. Determine whether user is calling about a shared mailbox or in individual mailbox.
  3. If shared mailbox, STOP. These instructions are for individual mailboxes ONLY. Return to knowledgebase and search for appropriate instructions.

  7. In the compose screen users are able to,

    • Request read receipts
    • Record or upload message audio
    • Send messages

How do I add recipients

back to top

  1. Gather resolution information and enter it into the ticket.
  2. Determine whether user is calling about a shared mailbox or in individual mailbox.
  3. If shared mailbox, STOP. These instructions are for individual mailboxes ONLY. Return to knowledgebase and search for appropriate instructions.
  4. Go to um.case.edu
  5. Enter CWRU network ID and password when prompted.
  6. Click New message to compose a new message.

  7. Add message recipients,

    Add recipients

    1. Begin typing recipient name in the the “To:” field.
    2. Select correct recipient from the list that appears.
    3. Repeat for additional recipients.

    Add Carbon copy (Cc) recipients

    1. Click the link labelled “Add Cc”.
    2. Begin typing recipient name in the the “Cc:” field.
    3. Select correct recipient from the list that appears.
    4. Repeat for additional Carbon copy recipients.

    Add Blind carbon copy (Bcc) recipients

    1. Click the link labelled “Add Bcc”.
    2. Begin typing recipient name in the the “Bcc:” field.
    3. Select correct recipient from the list that appears.
    4. Repeat for additional Blind carbon copy recipients.
  8. Resolve ticket.

Message responses

back to top

  1. Gather resolution information and enter it into the ticket.
  2. Determine whether user is calling about a shared mailbox or in individual mailbox.
  3. If shared mailbox, STOP. These instructions are for individual mailboxes ONLY. Return to knowledgebase and search for appropriate instructions.
  4. Go to um.case.edu
  5. Enter CWRU network ID and password when prompted.
  6. Select message by clicking anywhere on the message description. NOTE: When selected, a message is highlighted in green.

  7. Respond to, or delete selected message,

    Reply to a sender

    1. Click Reply.
    2. Address the reply (if additional recipients).
    3. Mark message options.
    4. Record message or upload message file.
    5. Click Send to send reply, or Cancel to abandon the message.

    Reply to sender and all recipients

    1. Click Reply all.
    2. Address the reply (if additional recipients).
    3. Mark message options.
    4. Record message or upload message file.
    5. Click Send to send reply, or Cancel to abandon the message.

    Forward message

    1. Click Reply to all.
    2. Address the message.
    3. Mark message options.
    4. Click Send to send reply, or Cancel to abandon the message.

    Delete message

    1. Click Delete. NOTE: Deleted messages are retained for 7 days prior to permanent delete.
  8. Resolve ticket
