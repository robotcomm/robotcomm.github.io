Troubleshoot + Repair

I Forgot My Password

back to top

  1. Gather resolution information and enter it into the ticket.
  2. Canvas uses SSO.
  3. Instruct user to sign on using CWRU network ID and password.
  4. Troubleshoot and resolve or assign to Identity Management resolver group as provided applicable script(s).
  5. Note troubleshooting steps taken in ticket comments.

Slow Response Time, Other Performance, Broken Links, etc.

back to top

  1. Gather resolution information and enter it into the ticket.
  2. Troubleshoot internet connectivity.
  3. Instruct the user to clear their cache. Explain how if necessary.
  4. Note troubleshooting steps taken in ticket comments.
  5. Resolve ticket, or Warm transfer customer to Instructure: 844-747-4614

User is Not Listed in a Class

back to top

  1. Gather resolution information and enter it into the ticket.
  2. Verify registration is complete and final in SIS.
  3. If user registration changes were completed within the last 24 hours, advise user to check again the next day. Resolve the ticket.
  4. If user registration changes were completed more than 24 hours ago, assign ticket to Canvas resolver group.
  5. Note troubleshooting steps taken in ticket comments.
