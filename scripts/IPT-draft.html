Identity Verification Steps

Please note, you must verify ALL users before providing any account information.

  1. Determine user role (e.g., student, field instructor).
  2. Students

    1. To verify, user should be able to correctly confirm without prompting:

      • Name
      • Network ID
      • CWRU email address (abc123@case.edu)
    2. If a student cannot independently verify this information, refer user to Henrientta Jones (hpj@case.edu) [MSASS].
    3. Do not provide account information.
    4. Resolve ticket, indicating that user identity is unverified.
  3. Field Instructors

    1. To verify, user should be able to correctly confirm without prompting:

      • Name
      • Birthdate
    2. If a field instructor cannot independently verify this information, refer user to Henrientta Jones (hpj@case.edu) [MSASS].
    3. Do not provide account information.
    4. Resolve ticket, indicating that user identity is unverified.
  4. If any user correctly verifies but does not have an IPT account, refer that person to Henrientta Jones (hpj@case.edu) [MSASS] and resolve ticket.
  5. After successful verification, proceed with troubleshooting steps.

Additional Information

Resolver Groups

IPT is not supported by a resolver group.

  • If an issue it outside the scope of support the Service Desk provides, the user should be directed to contact his or her field advisor, and the ticket closed.

How to Escalate

Intern Placement Tracking (IPT) is not considered a critical service. Escalation has not been made available at this time.

Troubleshoot + Repair

Reset a Lost Password (student)

back to top

  1. If not completed, verify user identity.
  2. Gather resolution information and enter it into the ticket.
  3. Verify that the user has attempted to reset his or her password already using the button marked "Forgot your username or password".
  4. If no, direct user to IPT login. Instruct user to perform self-service password reset.

    The user will need to provide an Organization ID (msass), User Type (student) and associated email address. After the user clicks Submit, the system will display the following confirmation: A reset password link has been emailed to you at abc12@case.edu. The link will be valid for the next two hours.

    Important For security reasons you should NOT provide students with any credential information over the phone that they have not confirmed first, including email addresses.

  5. Resolve the ticket, or, if unsuccessful, complete the following to reset the default password for the user.
  6. Sign in to IPT using either set of Administrator credentials:
    • Organization: msass
      Login Name: *******
      Password: *******
    • Organization: msass
      Login Name: *******
      Password: *******
  7. Click the tab marked Student List.
  8. Locate the student by name provided.
    list format: Last, First
  9. Click the name of the student to open a student detail page.
  10. Scroll to the bottom of the student detail page and select the box labeled "Reset Username and Password to Defaults".
  11. Click Save.
  12. The student will receive an email with a link and instructions to set a new password.
  13. If you cannot locate the user by name, he or she may need to have a new account created. Refer user to Henrientta Jones (hpj@case.edu) [MSASS].
  14. Resolve ticket.
