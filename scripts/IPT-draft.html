Intern Placement Tracking (IPT)
End User (MSASS) Support

Intern Placement Tracking (IPT) is used to student time spent in the field. The application is administered by staff and faculty (e.g., field advisors) in the Mandel School.

The ITS Service Desk will facilitate IPT password reset requests, only.

Reset a Lost Password

back to top

  1. Gather resolution information and enter it into the ticket.
  2. Verify that the user has attempted to reset his or her password already using the button marked "Forgot your username or password?
    3. .
  3. If no, direct user to IPT login. Instruct user to perform self-service password reset.

    The user will need to provide an Organization ID (msass), User Type (student) and associated email address. After the user clicks Submit, the system will display the following confirmation: A reset password link has been emailed to you at abc12@case.edu. The link will be valid for the next two hours.

    Important For security reasons you should NOT provide users with any credential information over the phone that they have not confirmed first, including email addresses.

    4. Resolve the ticket, or, if unsuccessful, complete the following to reset the default password for the user.

  4. Sign in to IPT using either set of Administrator credentials:
    • Organization: msass
      Login Name: *******
      Password: *******
    • Organization: msass
      Login Name: *******
      d Password: *******
  5. Click the tab marked Student List.
  6. Locate the student by name provided.
    list format: Last, First
  7. Click the name of the student to open a student detail page.
  8. Scroll to the bottom of the student detail page and select the box labeled "Reset Username and Password to Defaults".
  9. Click Save.
  10. The student will receive an email with a link and instructions to set a new password.
  11. Resolve ticket.
