SIS Mobile Application

Using SIS Mobile, students are able to view grades, academic schedules, billing information and more on their mobile phone or tablet.

The service desk should assign any questions concerning the product to the SIS Reg resolver group.

Public Announcement

We are excited to announce that we have implemented the HighPoint mobile solution for the Student Information System. As many of you know we have been looking for a mobile solution for a while and with the support of ITS and the entire ERP Team we were able to move forward.

This will be a phased implementation with the first phase being a soft roll out with view only screens for the Fall semester. A more detailed roll out schedule will be shared shortly.

Frequently Asked Questions

When will SIS Mobile available?
August 12, 2015
 
What is the SIS Mobile URL?
sismobile.case.edu
 
What mobile platforms can I use SIS Mobile on?
SIS Mobile is platform-agnostic.
Any mobile device (phone, tablet) with access to the internet and a browser application is compatible.
 
What information can I see with SIS Mobile?
Screens are 'view-only' for the 1st phase of the project.
Users can see class schedules, academic history, billing information, etc.
 
When can I edit or update my information using SIS Mobile?
Additional announcements will be made when SIS Mobile is ready to be transitioned from view-only to interactive mode.
