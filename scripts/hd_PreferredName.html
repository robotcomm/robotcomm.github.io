Service Description

Students and employees of CWRU may request that the name that appears in directory search be changed to suit professional requirements (e.g., employee prefers to be known professionally by unmarried name).

Preferred name changes will not affect email identity, CWRU network ID or any name that appears on a legal document (e.g., payroll, taxation, etc.)

NOTE: Request a preferred name change in person.

Employees must visit Human Resources, Crawford building, 3rd floor. Students may request name changes through the Registrar’s office, 135 Yost Hall.

What documents does a preferred name change affect?

return to index

  1. Gather resolution information and enter it into the ticket.
  2. Preferred name changes will not affect any documents that require legal name (e.g., paystubs, taxation info, etc.)
  3. Preferred name change will not change CWRU Network ID or email address.
  4. Preferred name change will change display name used in CWRU directory.
  5. Resolve the ticket.
