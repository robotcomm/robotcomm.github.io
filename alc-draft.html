Wildcard Search Values

Standard wildcard characters may be used in searches. Searches that return a large number records (e.g., Network ID equals *) will take a long time to complete and may cause the browser to time out.

Wildcard searches are most useful when you do not know if a name has an unusual spelling, pronunciation, or is long and may have been shortened in some records (for example Shannon Huntington Smythe-Whitely => Shannon Huntington). A search like Full name equals Shannon *huntington* would reduce the selection set considerably but still produce useful results.
